## 🔰 PHASE 0: COMPUTER & INTERNET FUNDAMENTALS
*Para sa mga baguhan talaga. Dito sisimulan ang tamang pundasyon.*

### Lesson 0.1 – Ano ang ICT at Bakit Ito Mahalaga?
- Definition ng Information and Communications Technology
- Mga karera sa ICT (tech support, programmer, network admin, cybersecurity, data analyst)
- Paano natin ginagamit ang ICT sa araw-araw
- **Taglish:** "ICT ang dahilan kung bakit ka nakakapag-Facebook, nakakapag-send ng email, at napapatakbo ang businesses online."

### Lesson 0.2 – Parts of a Computer (Hardware Basics)
- Input, Output, Processing, Storage devices
- CPU, RAM, Hard Drive/SSD, Motherboard, PSU
- Peripheral devices: keyboard, mouse, monitor, printer
- **Taglish:** "Ang CPU ang utak, RAM ang short-term memory, at ang hard drive ang long-term storage. Kapag mabagal PC mo, baka kulang sa RAM."

### Lesson 0.3 – Operating Systems (Windows, macOS, Linux)
- Ano ang OS at bakit kailangan?
- Interface navigation: desktop, taskbar, start menu, file explorer
- Basic settings: display, network, user accounts
- Linux overview (Ubuntu) — open-source alternative
- **Taglish:** "Windows ang pinakagamitin sa offices at schools, pero sa servers mas sikat ang Linux."

### Lesson 0.4 – File Management & Organization
- Files vs folders (directories)
- Common file extensions: `.docx`, `.pdf`, `.jpg`, `.exe`, `.zip`
- Copy, move, rename, delete, restore from Recycle Bin
- File Explorer navigation and shortcuts
- **Taglish:** "Mag-organize ng folders per project or per subject — promise, mas madali maghanap."

### Lesson 0.5 – Internet Basics & Web Browsers
- How the Internet works (simplified): ISP, DNS, servers
- Web browsers: Chrome, Firefox, Edge
- URLs, domains, and hyperlinks
- Browser tabs, bookmarks, history, incognito mode
- **Taglish:** "Kapag nag-type ka ng `google.com`, nagtatanong muna sa DNS 'Anong address nito?' bago pumunta sa server."

### Lesson 0.6 – Online Safety & Digital Citizenship
- Strong passwords (at password manager)
- Phishing, scams, at malwares
- Responsible social media use
- Privacy settings at personal information
- **Taglish:** "Kung may email na nagsasabing nanalo ka ng 1 million pero hindi ka sumali, scam 'yan. Huwag i-click ang link!"

## 🔧 PHASE 2: COMPUTER HARDWARE & TROUBLESHOOTING
*Para sa aspiring tech support o gusto lang maunawaan kung bakit nagha-hang ang PC.*

### Lesson 2.1 – Inside the System Unit
- Opening the case safely (grounding, ESD)
- Identifying components: motherboard, CPU, RAM slots, GPU, expansion slots
- Power supply unit (PSU) connectors
- **Taglish:** "Bago buksan ang PC, patayin at hugutin sa saksakan. Gumamit ng anti-static wrist strap kung meron."

### Lesson 2.2 – Processors (CPU) & Cooling
- CPU brands (Intel, AMD), cores, threads, clock speed
- CPU socket types
- Heatsink and fan installation, thermal paste
- Liquid cooling basics
- **Taglish:** "Kung nag-o-overheat ang PC, baka tuyo na ang thermal paste — palitan every 2-3 years."

### Lesson 2.3 – Memory (RAM) & Storage
- Types of RAM (DDR3, DDR4, DDR5), speed, capacity
- Installing RAM sticks (dual-channel)
- HDD vs SSD (SATA, NVMe M.2)
- Disk management: partitioning, formatting, defrag (for HDD), TRIM (for SSD)
- **Taglish:** "SSD sobrang bilis kumpara sa HDD — laking difference sa boot time. Kung naka-HDD ka pa, upgrade na!"

### Lesson 2.4 – Peripherals & Printers
- Keyboards (membrane vs mechanical), mice (optical vs laser)
- Monitor types (LCD, LED, IPS, refresh rate)
- Printer types: inkjet, laser, thermal
- Installing printer drivers, shared printers
- Basic printer troubleshooting (paper jam, ink/toner issues)

### Lesson 2.5 – BIOS/UEFI & Boot Process
- What is BIOS/UEFI? Accessing setup (DEL, F2, F10)
- Boot order, enabling virtualization, secure boot
- POST (Power-On Self-Test) beep codes
- CMOS battery
- **Taglish:** "Kung ayaw mag-boot, pumasok sa BIOS at icheck kung nade-detect ang hard drive."

### Lesson 2.6 – Common PC Problems & Troubleshooting
- No power, no display, beeping sounds
- Blue Screen of Death (BSOD) — error codes
- Slow performance — Task Manager (CPU, RAM, Disk usage)
- Safe Mode, System Restore, Windows Recovery
- **Taglish:** "Bago tawagin ang technician, subukan muna mag-restart — maraming problema nasosolve ng restart."

### Lesson 2.7 – Building a PC from Scratch (Guide)
- Compatibility checking (PC Part Picker)
- Step-by-step assembly (CPU, cooler, RAM, motherboard in case, PSU, GPU, cable management)
- First boot and OS installation
- **Taglish:** "Parang Lego ang PC building — masaya pero dapat compatible ang parts. Huwag pwersahin!"

## 🌐 PHASE 3: NETWORKING ESSENTIALS
*Matutunan kung paano nag-uusap ang mga computers sa isa't isa — backbone ng internet.*

### Lesson 3.1 – Introduction to Computer Networks
- What is a network? LAN, WAN, MAN
- Network topologies: star, bus, ring, mesh
- Client-server vs peer-to-peer
- **Taglish:** "Ang network parang kalsada — may daanan (cable/Wi-Fi), may address ang bawat bahay (IP address), may traffic rules (protocols)."

### Lesson 3.2 – IP Addressing & Subnetting (Basics)
- IPv4 vs IPv6
- Binary to decimal conversion
- IP address classes (A, B, C), private vs public IP
- Subnet mask and CIDR notation (e.g., /24)
- Basic subnetting calculations
- **Taglish:** "Ang subnet mask nagsasabi kung ilan ang pwedeng devices sa isang network. `255.255.255.0` means 254 usable hosts."

### Lesson 3.3 – Network Devices & Cabling
- Router, switch, hub (difference), modem, access point
- Ethernet cables: straight-through vs crossover, Cat5e, Cat6
- Fiber optic, coaxial
- Crimping RJ45 connectors (T568A/B standards)
- **Taglish:** "Gumagamit ng straight-through cable para PC-to-switch, crossover para PC-to-PC."

### Lesson 3.4 – The OSI Model & TCP/IP Model
- 7 layers of OSI (Please Do Not Throw Sausage Pizza Away)
- Functions of each layer (Application, Transport, Network, etc.)
- TCP/IP model (4 layers)
- Common protocols: HTTP, FTP, SMTP, DNS, DHCP, ARP
- **Taglish:** "Bawat layer may kanya-kanyang trabaho. Kapag nagb-browse ka, dumadaan sa lahat ng layer."

### Lesson 3.5 – DNS, DHCP & NAT
- DNS: domain name resolution (A, CNAME, MX records)
- DHCP: automatic IP assignment (lease, renewal)
- NAT: sharing one public IP to many private IPs
- Port forwarding
- **Taglish:** "Kapag nag-connect ka sa Wi-Fi, si DHCP magbibigay sa'yo ng IP. Si DNS ang maghahanap ng address ng website."

### Lesson 3.6 – Basic Network Troubleshooting Commands
- `ping`, `tracert` / `traceroute`
- `ipconfig` / `ifconfig`, `nslookup`
- `netstat`, `arp -a`
- Checking connectivity at latency
- **Taglish:** "Kapag walang internet, unang gawin: mag-ping sa 8.8.8.8 (Google DNS). Kapag may reply, ibig sabihin may connection pero baka DNS issue."

### Lesson 3.7 – Wi-Fi & Wireless Networking
- Wi-Fi standards (802.11a/b/g/n/ac/ax), frequencies (2.4GHz vs 5GHz)
- SSID, security (WPA2, WPA3)
- Channel interference, signal strength
- Setting up a home router (admin panel)
- **Taglish:** "Ang 2.4GHz mas malayo abot pero mas mabagal; ang 5GHz mabilis pero maikli range. Pumili ng tamang frequency."

### Lesson 3.8 – Introduction to Network Security
- Firewalls (software at hardware)
- VPN basics (privacy, tunneling)
- MAC filtering, disabling WPS
- Basic router security settings
- **Taglish:** "Palitan lagi ang default admin password ng router — madalas `admin/admin` lang 'yan."

## 👨‍💻 PHASE 4: INTRODUCTION TO PROGRAMMING & SCRIPTING
*Dito magsisimula kang magsulat ng code. Kahit hindi ka programmer, useful ito sa automation.*

### Lesson 4.1 – Programming Concepts 101
- Algorithm, flowchart, pseudocode
- Variables, data types (integer, string, boolean)
- Control structures: sequence, selection, iteration
- **Taglish:** "Ang programming ay pagbibigay ng sunod-sunod na utos sa computer gamit ang wikang naiintindihan niya."

### Lesson 4.2 – Introduction to Python (Beginner-Friendly)
- Installing Python, IDEs (VS Code, PyCharm, Thonny)
- `print()`, variables, input
- Conditionals: `if/elif/else`
- Loops: `for`, `while`
- **Taglish:** "Python ang pinakamadaling language para sa beginners — parang English magbasa."

### Lesson 4.3 – Functions & Modules in Python
- Defining functions: `def function_name():`
- Parameters and return
- Importing modules (`math`, `random`, `datetime`)
- Writing and using simple scripts
- **Taglish:** "Kapag paulit-ulit ang code, gawin function para isang tawag na lang."

### Lesson 4.4 – Working with Files & Automation
- Reading and writing text files
- Working with CSV files (using `csv` module)
- Automating boring stuff: renaming files, organizing folders
- Task scheduling (cron on Linux, Task Scheduler on Windows)

### Lesson 4.5 – Introduction to Bash/Shell Scripting
- Linux terminal basics (`ls`, `cd`, `cp`, `mv`, `grep`, `chmod`)
- Writing shell scripts: shebang, variables, conditionals
- Automating system tasks (backup, cleanup)
- **Taglish:** "Sa server administration, kailangan ang shell scripting — para auto backup ng database gabi-gabi."

### Lesson 4.6 – Version Control with Git & GitHub
- What is version control? Git concepts (repository, commit, branch)
- Installing Git, basic commands (`git init`, `git add`, `git commit`, `git push`)
- GitHub: creating repos, pull requests, README
- Collaboration basics
- **Taglish:** "Ang Git parang time machine — kung nagkamali ka, kaya mong bumalik sa nakaraang version ng code."

### Lesson 4.7 – Introduction to Markdown & Documentation
- Markdown syntax (headings, bold, lists, links, images, code blocks)
- Writing README files for projects
- Documentation tools (MkDocs, Docusaurus basics)
- **Taglish:** "Lahat ng ICT professional marunong mag-Markdown — simple lang pero sobrang linis ng documentation."

## 🌍 PHASE 5: WEB TECHNOLOGIES & DEVELOPMENT
*Para sa gustong gumawa ng websites at web applications.*

### Lesson 5.1 – How the Web Works
- HTTP/HTTPS, requests (GET, POST), responses (status codes)
- Domain names, hosting, DNS
- Frontend vs Backend
- **Taglish:** "Kapag nag-type ka ng URL, nagse-send ng request si browser, si server nagre-respond ng HTML na nire-render ng browser."

### Lesson 5.2 – HTML5 Fundamentals
- Document structure (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
- Text elements (headings, paragraphs, lists)
- Links, images, tables
- Semantic HTML: `<header>`, `<nav>`, `<main>`, `<footer>`, `<article>`
- Forms: input types, textarea, buttons
- **Taglish:** "HTML ang skeleton ng website — nagbibigay ng structure."

### Lesson 5.3 – CSS3 Fundamentals & Responsive Design
- Selectors, properties, values
- Box model (margin, border, padding, content)
- Flexbox and CSS Grid (layout)
- Media queries for responsive design (mobile-friendly)
- Colors, typography, backgrounds
- **Taglish:** "CSS ang nagpapaganda ng website. Gamitin ang Flexbox para hindi na mahirapan sa pag-a-align."

### Lesson 5.4 – JavaScript Essentials for the Web
- Variables, data types, functions (crash recap)
- DOM manipulation (`getElementById`, `querySelector`)
- Events (click, submit, keypress)
- Fetch API (calling backend APIs)
- **Taglish:** "JavaScript ang nagbibigay buhay sa website — kapag may click, may animation, may pop-up."

### Lesson 5.5 – Introduction to Backend Development (PHP or Node.js)
- Server-side scripting, handling requests
- PHP basics: variables, forms, sessions (PHP path) *or* Node.js with Express basics
- Connecting to a database (MySQL)
- Creating a simple login system
- **Taglish:** "Backend ang nagpro-proseso ng data at nag-i-store sa database — hindi ito nakikita ng user."

### Lesson 5.6 – Databases for the Web (MySQL/SQLite)
- Creating databases and tables (phpMyAdmin or CLI)
- CRUD operations (Create, Read, Update, Delete)
- Using PDO or mysqli in PHP (or SQLite3 in Python/Node)
- Relationships (one-to-many)
- **Taglish:** "Ang database parang Excel file na sobrang laki at mabilis maghanap — dito naka-store ang users, posts, products."

### Lesson 5.7 – Introduction to CMS (WordPress)
- What is a CMS? Installing WordPress (XAMPP/Local)
- Themes, plugins, widgets
- Creating pages, posts, menus
- Basic customization
- **Taglish:** "WordPress — 43% ng websites sa buong mundo! Kahit hindi marunong mag-code, kayang gumawa ng site."

### Lesson 5.8 – Web Hosting & Deployment
- Choosing a hosting provider (shared, VPS, cloud)
- Uploading files via FTP/cPanel
- Domain setup, DNS pointing
- Deploying a static site (Netlify/GitHub Pages) or dynamic site (Heroku, 000webhost)
- SSL certificates (Let's Encrypt)
- **Taglish:** "Matapos mo gawin ang site, kailangan mong i-deploy para makita ng buong mundo. Maraming libreng hosting para sa practice."

## 🗃️ PHASE 6: DATABASES & DATA MANAGEMENT
*Para sa mga gustong mag-focus sa data — storage, retrieval, at analysis.*

### Lesson 6.1 – Database Fundamentals
- What is a database? DBMS (MySQL, PostgreSQL, SQLite)
- Relational vs NoSQL databases (MongoDB example)
- Tables, records, fields, primary keys, foreign keys
- **Taglish:** "Ang relational database naka-organize sa tables na parang mga spreadsheet na may relasyon sa isa't isa."

### Lesson 6.2 – SQL Basics (Structured Query Language)
- `SELECT`, `WHERE`, `ORDER BY`, `LIMIT`
- `INSERT INTO`, `UPDATE`, `DELETE`
- Filtering with `LIKE`, `BETWEEN`, `IN`
- Aggregate functions: `COUNT`, `SUM`, `AVG`, `MIN`, `MAX`
- **Taglish:** "Ang SQL ang wika ng database — ginagamit para magtanong (`SELECT`) o mag-utos (`UPDATE`)."

### Lesson 6.3 – Advanced SQL: Joins & Subqueries
- `INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`
- Table aliases, self-join
- Subqueries (nested SELECT)
- `GROUP BY` at `HAVING`
- **Taglish:** "Kapag gusto mo pagsamahin ang data mula sa dalawang table, gumamit ng JOIN — i-link gamit ang foreign key."

### Lesson 6.4 – Database Design & Normalization
- Entity-Relationship Diagrams (ERD)
- Normalization (1NF, 2NF, 3NF)
- Denormalization trade-offs
- Designing a database schema for a small application
- **Taglish:** "Huwag maglagay ng paulit-ulit na data — i-normalize para iwas redundancy at inconsistency."

### Lesson 6.5 – Working with Databases in Backend Code
- Connecting to MySQL using PHP PDO (or Python sqlite3)
- Prepared statements (preventing SQL injection)
- Handling transactions
- Pagination and search queries
- **Taglish:** "Never concatenate user input sa SQL query — laging gumamit ng prepared statements."

### Lesson 6.6 – Introduction to Data Analysis & Reporting
- Exporting data to CSV/Excel
- Simple data analysis using Python (Pandas) or Excel pivot tables
- Visualizing data (charts, dashboards) using tools like Google Data Studio (Looker) or Tableau Public
- **Taglish:** "Hindi lang storage — dapat marunong ka ring mag-analyze ng data para maging useful sa decision-making."

## 🔐 PHASE 7: CYBERSECURITY FUNDAMENTALS
*Napakahalaga ng security sa kahit anong ICT role.*

### Lesson 7.1 – Introduction to Cybersecurity
- CIA Triad: Confidentiality, Integrity, Availability
- Common threats: malware, ransomware, phishing, social engineering
- Security layers (defense in depth)
- **Taglish:** "Cybersecurity ay hindi lang antivirus — kasama na rito ang pag-iingat ng user at tamang configuration."

### Lesson 7.2 – Malware Types & Prevention
- Virus, worm, trojan, spyware, adware, ransomware
- How malware spreads (email attachments, downloads, USB drives)
- Anti-malware software, real-time protection
- Safe browsing habits

### Lesson 7.3 – Network Security Basics
- Firewalls (Windows Firewall, iptables basics)
- Intrusion Detection/Prevention Systems (IDS/IPS) concept
- Securing Wi-Fi (WPA3, disabling WPS, guest network)
- VPN: how it works, when to use
- **Taglish:** "Ang firewall ay gatekeeper — sinasala nito kung anong traffic papasok o lalabas ng network."

### Lesson 7.4 – Authentication & Access Control
- Multi-Factor Authentication (MFA)
- Password policies (complexity, expiry, no reuse)
- Principle of least privilege
- Role-Based Access Control (RBAC)
- **Taglish:** "Gumamit ng password manager at i-enable ang 2-factor authentication sa lahat ng important accounts."

### Lesson 7.5 – Web Application Security (OWASP Top 10)
- SQL Injection (prevention with parameterized queries)
- Cross-Site Scripting (XSS) — escaping output
- Cross-Site Request Forgery (CSRF)
- Sensitive data exposure, security misconfiguration
- **Taglish:** "Kung ikaw ay web developer, dapat alam mo ang OWASP Top 10 — checklist 'yan ng dapat iwasang vulnerability."

### Lesson 7.6 – Encryption & Data Protection
- Symmetric vs asymmetric encryption
- Hashing (SHA-256, bcrypt) for passwords
- SSL/TLS certificates (HTTPS)
- Full disk encryption (BitLocker, FileVault)

### Lesson 7.7 – Incident Response & Ethical Hacking Intro
- Incident response steps: preparation, detection, containment, eradication, recovery
- Introduction to penetration testing (ethical hacking)
- Tools: Kali Linux, Wireshark, Nmap (overview)
- Legal and ethical considerations
- **Taglish:** "Ang ethical hacker ay 'good guy' — sinusubukan pasukin ang system para malaman ang butas at maayos bago pasukin ng totoong attacker."

## ☁️ PHASE 8: CLOUD COMPUTING & EMERGING TECH
*Para updated ka sa latest technology trends.*

### Lesson 8.1 – Cloud Computing Concepts
- What is the cloud? IaaS, PaaS, SaaS
- Public, private, hybrid cloud
- Benefits: scalability, cost-efficiency, accessibility
- **Taglish:** "Imbes na bibili ka ng sariling server, nagre-rent ka ng computing resources sa cloud (e.g., AWS, Google Cloud, Azure)."

### Lesson 8.2 – Cloud Service Providers (AWS, Google Cloud, Azure)
- Overview of major cloud platforms
- Core services: compute (EC2, VMs), storage (S3, Blob), databases (RDS)
- Free tier accounts for learning
- Creating a virtual machine in the cloud

### Lesson 8.3 – Virtualization & Containers
- Virtual machines (VirtualBox, VMware)
- Containers: Docker basics (images, containers, Dockerfile)
- Difference between VMs and containers
- **Taglish:** "Container parang kapsula na may lahat ng kailangan ng application — consistent sa kahit anong environment."

### Lesson 8.4 – Internet of Things (IoT)
- What is IoT? Smart devices, sensors
- IoT architecture (device, gateway, cloud)
- Examples: smart home, agriculture, health
- Basic IoT project (Arduino or Raspberry Pi intro)
- **Taglish:** "IoT — pati ref, ilaw, at sasakyan nakakonekta na sa internet. Daming opportunities dito."

### Lesson 8.5 – AI, Machine Learning & Automation Basics
- AI vs ML vs Deep Learning
- Applications: chatbots, image recognition, recommendation systems
- No-code/low-code AI tools (Teachable Machine, ChatGPT API)
- Automation using RPA (Robotic Process Automation) concept
- **Taglish:** "Hindi mo kailangang maging data scientist para gumamit ng AI — maraming APIs na pwedeng i-integrate."

## 📚 RECOMMENDED CAPSTONE PROJECTS (Pumili ng isa)

4. **Personal Portfolio & Blog** (Static site with CMS integration) – Frontend & backend skills.
5. **Cybersecurity Awareness App** (Interactive website or mobile-friendly app with quizzes, tips) – Shows security knowledge and web skills.
