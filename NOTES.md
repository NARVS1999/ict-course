# Notes

## User Preferences
- Medium: Taglish (Tagalog + English)
- Want: Structured, sequential learning path
- Goal: Maging competent ICT professional
- Free resources only
- Hands-on learning with quizzes and mini-projects
- Self-paced

## Teaching Approach
- Each lesson: Knowledge first, then skill practice
- Include Taglish explanations for key concepts
- Quizzes for retrieval practice
- Mini-projects for hands-on application
- Reference to external high-quality resources
- Sequential build-up (bawat phase isang hakbang)
