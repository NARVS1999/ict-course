HTML & CSS Cheatsheet

HTML Structure

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>Heading</h1>
    <p>Paragraph</p>
</body>
</html>

Common HTML Elements

TagPurpose
<h1> to <h6>Headings (1 largest, 6 smallest)
<p>Paragraph
<a href="...">Link
<img src="...">Image
<ul> / <ol>Unordered / Ordered list
<li>List item
<table>Table
<form>Form for user input
<input>Input field
<div>Generic container (block)
<span>Generic container (inline)

Semantic HTML5

<header>  — site header / intro
<nav>     — navigation links
<main>    — main content (isa lang per page)
<article> — independent content
<section> — thematic group
<aside>   — sidebar / related content
<footer>  — site footer

CSS Selectors

tagname { }         — select by tag: p { }
.class-name { }     — select by class: .highlight { }
#id-name { }        — select by id: #header { }
tag.class { }       — tag with class: p.special { }
parent child { }    — descendant: div p { }
parent > child { }  — direct child: ul > li { }

CSS Box Model

element {
    margin: 10px;       /* space outside border */
    border: 1px solid;  /* border */
    padding: 10px;      /* space inside border */
    width: 200px;       /* content width */
    height: 100px;      /* content height */
}

Flexbox

/* Parent container */
.container {
    display: flex;
    justify-content: center; /* horizontal: flex-start, center, space-between */
    align-items: center;     /* vertical: flex-start, center, stretch */
    gap: 10px;               /* space between items */
    flex-wrap: wrap;         /* allow wrapping */
}

/* Child items */
.item {
    flex: 1;  /* grow equally */
}

CSS Grid

.grid {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr; /* 3 equal columns */
    gap: 20px;
}
