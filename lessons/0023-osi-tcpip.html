Lesson 0023: The OSI Model & TCP/IP Model

Unawain ang 7 layers ng OSI model at ang 4 layers ng TCP/IP model — ang pundasyon ng networking.

Lesson Goal Malaman ang bawat layer ng OSI at TCP/IP models at kung anong protocols ang gumagana sa bawat layer.

OSI Model — 7 Layers

Ang OSI (Open Systems Interconnection) model ay isang konseptwal na framework na nagpapaliwanag kung paano gumagana ang network communication. 7 layers ito, mula sa pinakamababa hanggang sa pinakamataas: Physical (cables, bits), Data Link (MAC addresses, switches), Network (IP addressing, routers), Transport (TCP/UDP, port numbers), Session (establish/manage connections), Presentation (encryption, data formatting), Application (HTTP, FTP, SMTP — kung saan nakikipag-ugnayan ang user).

Mnemonic: Please Do Not Throw Sausage Pizza Away (Physical, Data Link, Network, Transport, Session, Presentation, Application).

Tracks: Ano ang Nangyayari sa Bawat Layer

Kapag nag-browse ka ng website: ang Application layer ay nagpapadala ng HTTP request. Ang Transport layer (TCP) ay nagha-hati ng data sa segments at tinitiyak na dumating ito nang maayos. Ang Network layer (IP) ay nag-a-add ng source at destination IP addresses. Ang Data Link layer ay nag-a-add ng MAC addresses para sa local delivery. Ang Physical layer ang nagpapadala ng actual bits sa cable o Wi-Fi.

"Bawat layer may kanya-kanyang trabaho. Kapag nagb-browse ka, dumadaan sa lahat ng layer."

TCP/IP Model — 4 Layers

Ang TCP/IP model ay ang practical na version ng OSI — 4 layers lang: Network Interface (Physical + Data Link), Internet (Network layer — IP), Transport (TCP, UDP), Application (Session + Presentation + Application — HTTP, FTP, DNS, DHCP, SMTP). Ito ang actual na ginagamit ng internet.

Mga Common Protocols

  • HTTP/HTTPS — web browsing (port 80/443)
  • FTP — file transfer (port 21)
  • SMTP — email sending (port 25)
  • DNS — domain name to IP (port 53)
  • DHCP — automatic IP assignment (port 67/68)
  • ARP — IP to MAC address resolution

Mini-Project: Memorize ang OSI Layers

Gamit ang mnemonic na "Please Do Not Throw Sausage Pizza Away" (Physical, Data Link, Network, Transport, Session, Presentation, Application), isaulo ang 7 layers ng OSI model. Isulat ang mga ito sa isang papel nang paulit-ulit hanggang sa kabisado mo. Pagkatapos, ipaliwanag sa sarili mong salita kung ano ang ginagawa ng bawat layer. I-record ang iyong sarili kung kinakailangan.

Summary

  • OSI model ay may 7 layers: Physical, Data Link, Network, Transport, Session, Presentation, Application.
  • TCP/IP model ay may 4 layers: Network Interface, Internet, Transport, Application.
  • HTTP ay nasa Application layer; TCP ay nasa Transport layer; IP ay nasa Network layer.
  • Ang bawat layer ay may specific na function — sama-sama silang nagtatrabaho para sa network communication.
