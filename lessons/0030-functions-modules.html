Lesson 0030: Functions & Modules

Matuto gumawa ng reusable code gamit ang functions at pag-import ng modules sa Python.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakagawa ng sarili mong functions at makakagamit ng built-in modules tulad ng math, random, at datetime.

Defining Functions

Gamit ang def keyword, pwedeng gumawa ng function na pwedeng tawagin kahit saan sa code. Ang return naman ay nagbabalik ng value.

"Kapag paulit-ulit ang code, gawin function para isang tawag na lang."
def greet(name):
    """Mag-print ng greeting."""
    return "Hello, " + name + "!"

message = greet("Maria")
print(message)  # Output: Hello, Maria!

# Function na may parameters at return value
def add(a, b):
    return a + b

result = add(5, 3)
print("Sum:", result)  # Output: Sum: 8

Importing Modules

Ang modules ay mga Python file na may reusable code. Gamitin ang import para magamit ang mga ito.

import math
import random
from datetime import datetime

# Math module
print(math.sqrt(16))       # 4.0
print(math.pi)             # 3.14159...

# Random module
print(random.randint(1, 10))  # Random number 1-10
print(random.choice(["red", "blue", "green"]))  # Random choice

# Datetime module
now = datetime.now()
print(now.strftime("%Y-%m-%d %H:%M:%S"))

Writing Simple Scripts

Pagsamahin ang functions at modules sa isang script file (.py).

# calculator.py
import math

def circle_area(radius):
    return math.pi * radius ** 2

def circle_circumference(radius):
    return 2 * math.pi * radius

r = float(input("Enter radius: "))
print(f"Area: {circle_area(r):.2f}")
print(f"Circumference: {circle_circumference(r):.2f}")
