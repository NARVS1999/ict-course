Lesson 0022: Network Devices & Cabling

Tuklasin ang mga network devices — router, switch, hub, modem, access point — at ang mga Ethernet cables at fiber optics.

Lesson Goal Matukoy ang pagkakaiba ng mga network devices at maunawaan ang mga uri ng network cables at ang kanilang gamit.

Network Devices: Router, Switch, Hub, Modem, Access Point

Router — nagkokonekta ng iba't ibang networks (hal. LAN to internet) at nagfa-forward ng data packets. Switch — nagkokonekta ng multiple devices sa iisang LAN at nagse-send ng data sa tamang device lang (mas matalino kaysa hub). Hub — tulad ng switch pero nagbo-broadcast ng data sa lahat ng ports (luma at inefficient). Modem — nagco-convert ng signal mula ISP (cable, DSL, fiber) papunta sa Ethernet. Access Point — nagbibigay ng Wi-Fi connectivity sa wired network.

Ethernet Cables: Straight-Through vs Crossover

Ang straight-through cable (T568A both ends o T568B both ends) ay ginagamit para sa PC-to-switch o router-to-switch connections. Ang crossover cable (T568A isang end, T568B kabilang end) ay ginagamit para sa PC-to-PC o switch-to-switch connections — pero sa modern devices, hindi na kailangan ang crossover dahil auto-MDIX na ang mga switch. Cat5e — hanggang 1 Gbps, Cat6 — hanggang 10 Gbps (mas mahigpit ang specs), Cat6a — 10 Gbps sa mas mahabang distansya.

"Gumagamit ng straight-through cable para PC-to-switch, crossover para PC-to-PC."

Fiber Optic at Coaxial Cables

Ang fiber optic ay gumagamit ng liwanag (light pulses) sa halip na electrical signals — sobrang bilis (hanggang 100+ Gbps), walang interference, at pwede sa malalayong distansya. Ito ang ginagamit ng mga ISPs para sa high-speed internet. Ang coaxial cable ay ang lumang uri na ginagamit sa cable TV at DSL internet — mas mabagal at mas madaling kapitan ng interference kumpara sa fiber.

RJ45 Crimping (T568A/T568B)

Ang RJ45 connector ang plug na nasa dulo ng Ethernet cable. Ang T568A at T568B ay ang dalawang wiring standards — pareho lang gumagana, pero sa Pilipinas, mas common ang T568B. Ang wire order ng T568B: White-Orange, Orange, White-Green, Blue, White-Blue, Green, White-Brown, Brown.

Mini-Project: Kilalanin ang Iyong Home Network Devices

Tingnan ang iyong setup sa bahay o sa school. Hanapin ang router, modem, switch (kung meron), access point, at iba pang network devices. I-drawing ang diagram kung paano sila naka-connect sa isa't isa. I-note ang brand at model ng bawat device. Kung may Ethernet cable, tingnan kung may nakasulat na "Cat5e" o "Cat6" sa cable.

Summary

  • Router = nagkokonekta ng networks; Switch = nagkokonekta ng devices sa LAN; Hub = luma at inefficient.
  • Straight-through cable para sa PC-to-switch; crossover cable para sa PC-to-PC.
  • Cat5e (1 Gbps) at Cat6 (10 Gbps) ang common Ethernet cable categories.
  • Fiber optic ang pinakamabilis at ginagamit ng ISPs para sa high-speed internet.
