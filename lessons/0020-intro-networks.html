Lesson 0020: Introduction to Computer Networks

Alamin ang konsepto ng computer networks — LAN, WAN, topologies, at client-server architecture.

Lesson Goal Maunawaan ang pangunahing konsepto ng networking at ang iba't ibang uri ng network topologies.

Ano ang Network?

Ang computer network ay dalawa o higit pang computers na magkaka-connect para mag-share ng resources (files, printers, internet). Ang pinakasimpleng network ay dalawang computer na naka-cable o naka-Wi-Fi. Ang LAN (Local Area Network) ay nasa iisang lugar lang (bahay, opisina, school). Ang WAN (Wide Area Network) ay sumasaklaw sa malawak na lugar — ang internet mismo ay ang pinakamalaking WAN. Ang MAN (Metropolitan Area Network) naman ay nasa pagitan — isang lungsod o campus.

Network Topologies

Ang topology ay ang physical o logical na arrangement ng devices sa network. Star — lahat ng devices ay naka-connect sa central device (switch/router) — ito ang pinaka-common. Bus — iisang cable lang, lahat ng devices naka-tap dito (luma na). Ring — bawat device ay naka-connect sa dalawang katabi, parang circle. Mesh — lahat ng devices ay interconnected — sobrang reliable pero mahal.

"Ang network parang kalsada — may daanan (cable/Wi-Fi), may address ang bawat bahay (IP address), may traffic rules (protocols)."

Client-Server vs Peer-to-Peer

Sa client-server model, may central server na nagbibigay ng resources (files, email, web pages) at clients na humihingi ng access — ito ang gamit sa mga opisina at sa internet. Sa peer-to-peer (P2P), lahat ng computers ay pantay-pantay at pwedeng mag-share ng files direkta sa isa't isa — walang central server — ito ay gamit sa maliliit na networks o file sharing apps.

Quick Quiz

1. Ano ang LAN?
2. Aling topology ang gumagamit ng central device (switch/router)?
3. Ano ang client-server model?

Mini-Project: Diagram ng Iyong Home Network

Gumuhit ng network diagram ng iyong setup sa bahay. Isama ang router, modem, computers, phones, printers, at iba pang devices. Tukuyin kung anong topology ang gamit ninyo (most likely star). Lagyan ng label ang bawat device at kung paano sila naka-connect (Wi-Fi o Ethernet). Pwedeng sa papel o gamit ang drawing software.

Summary

  • Ang network ay para sa pag-share ng resources sa pagitan ng computers.
  • LAN ay local, WAN ay malawak (internet), MAN ay nasa pagitan.
  • Star topology ang pinaka-common — lahat ng devices naka-connect sa central switch/router.
  • Client-server ay may central server; peer-to-peer ay walang central server.
