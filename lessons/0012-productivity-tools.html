Lesson 0012: Productivity Tools & Keyboard Shortcuts

Mga keyboard shortcuts at productivity tools para maging mas efficient sa computer.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makaka-memorize ng essential keyboard shortcuts, makagamit ng task management tools, at maia-apply ang Pomodoro technique.

Essential Keyboard Shortcuts

Ang keyboard shortcuts ay key combinations na mabilis na nagsasagawa ng command nang hindi ginagamit ang mouse. Narito ang mga essential:

  • Ctrl+C — Copy
  • Ctrl+V — Paste
  • Ctrl+Z — Undo (ibalik ang ginawang pagbabago)
  • Ctrl+A — Select All
  • Alt+Tab — Lumipat sa ibang open window
  • Win+D — Ipakita ang desktop (minimize lahat ng windows)
  • Win+E — Buksan ang File Explorer
  • Ctrl+Shift+Esc — Buksan ang Task Manager
"Matuto ng keyboard shortcuts — grabe ang tipid sa oras. Parang secret move sa fighting game."

Task Management Apps

Ang task management apps ay tumutulong sa iyo na mag-organize ng iyong mga gawain:

  • Trello — Visual boards na may cards para sa tasks. Parang sticky notes sa whiteboard.
  • Todoist — Simple list-based task manager na may due dates at priorities.
  • Google Keep — Quick notes na may reminders at color coding.
  • Notion — All-in-one workspace para sa notes, tasks, databases, at wikis.

Note-Taking: OneNote at Notion

Ang OneNote ay bahagi ng Microsoft Office at maganda para sa structured notes na may sections at pages. Ang Notion naman ay mas flexible — pwedeng gawing notes, task board, o kahit database.

Pomodoro Technique

Ang Pomodoro Technique ay isang time management method kung saan nagtatrabaho ka nang nakatutok sa loob ng 25 minuto (isang "pomodoro"), followed by 5-minute break. Pagkatapos ng 4 na pomodoros, mag-30 minutong mahabang break. Makakatulong ito para maiwasan ang burnout at mapanatili ang focus.

Mini-Project: Set Up a Trello Board

Gumawa ng Trello board para sa iyong weekly tasks. Gumawa ng mga columns tulad ng "To Do", "Doing", at "Done". Magdagdag ng cards para sa bawat task na kailangan mong gawin ngayong linggo. I-set ang due dates at labels kung kinakailangan.

Summary

  • Ang keyboard shortcuts tulad ng Ctrl+C, Ctrl+V, at Ctrl+Z ay nakakatipid ng oras.
  • Gamitin ang Trello, Todoist, o Google Keep para sa task management.
  • Ang Pomodoro technique ay 25 minuto work, 5 minuto break para sa mas epektibong pag-aaral o pagtatrabaho.
