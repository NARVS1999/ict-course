Lesson 0003: Operating Systems (Windows, macOS, Linux)

Pag-unawa sa operating system at pagkakaiba ng Windows, macOS, at Linux.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakapagpaliwanag kung ano ang OS, makilala ang Windows/macOS/Linux interfaces, at mag-navigate sa basic settings.

Ano ang Operating System?

Ang Operating System (OS) ay ang pangunahing software na nagmamanage ng computer hardware at nagbibigay ng interface para sa user. Ito ang nagpapatakbo ng lahat ng programs at nag-aallocate ng resources tulad ng memory, processor time, at storage. Kung walang OS, hindi mo magagamit ang computer.

"Windows ang pinakagamitin sa offices at schools, pero sa servers mas sikat ang Linux."

Windows vs macOS vs Linux

  • Windows — Gawa ng Microsoft. Pinakasikat sa mga offices, schools, at personal computers. User-friendly at maraming supported software at games.
  • macOS — Gawa ng Apple. Exclusive sa Mac computers. Known for its sleek design, stability, at security. Madalas gamit ng mga designers at creatives.
  • Linux (Ubuntu) — Open-source at libre. Maraming variety (distributions o "distros"). Mas gamit sa servers, developers, at advanced users. Ang Ubuntu ang pinaka-user-friendly na version ng Linux.

Basic Settings

Sa Windows, makikita mo ang Settings app (gear icon) kung saan mo pwedeng i-customize ang Display, Network, User Accounts, at iba pa. Sa macOS, nasa System Settings ito. Sa Linux (Ubuntu), nasa Settings app din ang mga basic configuration.

Mini-Project: Explore Windows Settings

Buksan ang Settings app sa Windows (Start menu > gear icon). I-explore ang mga sumusunod: Display settings, Network & Internet, at Accounts. Isulat sa notebook kung ano ang mga options na makikita mo sa bawat category.

Summary

  • Ang OS ay nagmamanage ng hardware at nagbibigay ng interface sa user.
  • Windows ang pinakagamit sa offices/schools; Linux ang mas sikat sa servers.
  • Ubuntu ang pinaka-beginner-friendly na Linux distribution.
