Lesson 0027: Introduction to Network Security

Alamin ang basics ng network security — firewalls, VPN, MAC filtering, at kung paano i-secure ang iyong home router.

Lesson Goal Maunawaan ang mga basic security measures para protektahan ang network — firewalls, VPN, at router security settings.

Firewalls: Software at Hardware

Ang firewall ay parang guard ng network — sinusuri nito ang incoming at outgoing traffic at binablock ang suspicious connections. May software firewall (naka-install sa computer tulad ng Windows Defender Firewall) at hardware firewall (built-in sa router). Siguraduhing naka-enable ang firewall sa router at sa bawat computer para sa double protection.

VPN (Virtual Private Network)

Ang VPN ay gumagawa ng naka-encrypt na tunnel sa pagitan ng computer at ng VPN server. Pinoprotektahan nito ang iyong privacy sa pamamagitan ng pagtatago ng iyong totoong IP address at pag-e-encrypt ng iyong internet traffic — lalo na kapag gumagamit ng public Wi-Fi (sa coffee shop, airport). Ang VPN ay nagbibigay ng privacy at security sa pamamagitan ng tunneling (pag-encapsulate ng data).

MAC Filtering at WPS

Ang MAC filtering ay isang security feature na nagpapahintulot lamang sa specific devices (base sa MAC address) na kumonekta sa Wi-Fi. Pero hindi ito 100% secure dahil pwedeng i-spoof ang MAC address. Ang WPS (Wi-Fi Protected Setup) ay isang feature na nagpapadali sa pagkonekta (pindot lang ng button o mag-type ng PIN) — pero may security vulnerability ito, kaya i-disable ang WPS para mas secure ang network.

"Palitan lagi ang default admin password ng router — madalas admin/admin lang 'yan."

Basic Router Security Settings

  • Palitan ang default admin password — huwad na iwanan ang "admin/admin".
  • Palitan ang SSID — huwag gamitin ang default na pangalan na nagpapakita ng brand ng router.
  • I-enable ang WPA2 o WPA3 — huwag gumamit ng WEP o open network.
  • I-disable ang remote management — para walang makapasok sa router settings mula sa internet.
  • I-update ang router firmware — para ayusin ang mga security vulnerabilities.

Mini-Project: Check Your Router Security Settings

Kung maa-access mo ang router admin panel, i-log in (karaniwang 192.168.1.1 o 192.168.0.1 sa browser). Hanapin ang security settings at i-check ang mga sumusunod: naka-WPA2/WPA3 ba ang Wi-Fi? Naka-disable ba ang WPS? Naka-disable ba ang remote management? Kung hindi mo ma-access, manood ng YouTube video kung paano i-secure ang isang home router at i-document ang steps.

Summary

  • Firewall ang nagbabantay sa network traffic — dapat naka-enable sa router at sa computer.
  • VPN ay nag-e-encrypt ng traffic para sa privacy — importante sa public Wi-Fi.
  • WPS ay may security risk — i-disable ito sa router settings.
  • Palitan ang default admin password ng router at i-update ang firmware regularly.
