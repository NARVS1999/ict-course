Lesson 0049: Introduction to Cybersecurity

Pag-unawa sa mga pangunahing konsepto ng cybersecurity: CIA Triad, common threats, at defense in depth.

Lesson Goal Matutunan ang fundamentals ng cybersecurity — kung paano protektahan ang data, kilalanin ang mga banta, at magpatupad ng security layers.

CIA Triad

Ang CIA Triad ay ang pundasyon ng cybersecurity:

  • Confidentiality — Tanging ang awtorisadong tao lamang ang may access sa data. Halimbawa: encryption, passwords.
  • Integrity — Hindi nababago ang data ng unauthorized na tao. Halimbawa: hashing, checksums.
  • Availability — Ang data at system ay accessible kapag kailangan. Halimbawa: backups, redundancy.

Common Threats

Malware — Malicious software (virus, worm, trojan). Ransomware — Nagla-lock ng files at humihingi ng ransom. Phishing — Pekeng email/message na kunwari ay galing sa lehitimong source para nakawin ang credentials. Social Engineering — Pagmamanipula ng tao para magbigay ng kumpidensyal na impormasyon.

Defense in Depth (Security Layers)

Hindi sapat ang isang depensa lang. Ang defense in depth ay maraming layers: firewall, antivirus, strong passwords, encryption, user training, at regular updates. Kung bumagsak ang isang layer, may iba pang depensa.

"Cybersecurity ay hindi lang antivirus — kasama na rito ang pag-iingat ng user at tamang configuration."

Mini-Project: Identify 3 Security Measures

Maglista ng tatlong security measures na ginagamit mo araw-araw. Halimbawa: pagla-lock ng phone, paggamit ng password, pag-iiwas sa pag-click ng suspicious links. Ipaliwanag kung paano nakatutulong ang bawat isa sa iyong seguridad.

Summary

  • CIA Triad: Confidentiality, Integrity, Availability — pundasyon ng cybersecurity.
  • Mga common threats: malware, ransomware, phishing, social engineering.
  • Defense in depth = maraming layers ng depensa para sa mas matibay na proteksyon.
