Lesson 0031: Working with Files & Automation

Matutong magbasa at magsulat ng files sa Python, at mag-automate ng mga paulit-ulit na gawain.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakapagbasa at makakapagsulat ng text at CSV files, at makakagawa ng automation script para sa file organization.

Reading and Writing Text Files

Gamit ang open() function, pwedeng magbasa at magsulat ng files. May iba't ibang modes: 'r' para read, 'w' para write, 'a' para append.

"Kapag manual mong ginagawa ang paulit-ulit na tasks sa computer, malamang may Python script na pwedeng mag-automate nito. Mas masarap matulog habang tumatakbo ang script mo."
# Pagsulat ng file
with open("sample.txt", "w") as file:
    file.write("Hello, File!\n")
    file.write("Ito ay line 2.\n")

# Pagbasa ng file
with open("sample.txt", "r") as file:
    content = file.read()
    print(content)

# Magbasa linya por linya
with open("sample.txt", "r") as file:
    for line in file:
        print(line.strip())

Working with CSV Files

Ang CSV (Comma-Separated Values) ay common na format para sa data tables. Gamitin ang csv module.

import csv

# Pagsulat ng CSV
with open("students.csv", "w", newline="") as file:
    writer = csv.writer(file)
    writer.writerow(["Name", "Age", "Course"])
    writer.writerow(["Juan", 20, "BSIT"])
    writer.writerow(["Maria", 19, "BSCS"])

# Pagbasa ng CSV
with open("students.csv", "r") as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

Automating Boring Stuff

Pwedeng mag-automate ng mga bagay tulad ng pag-rename ng files, pag-oorganize ng folders, at pag-backup.

import os
import shutil

# Mag-rename ng files sa isang folder
folder = "downloads"
for filename in os.listdir(folder):
    if filename.endswith(".jpg"):
        new_name = "photo_" + filename
        os.rename(os.path.join(folder, filename),
                  os.path.join(folder, new_name))

# Backup ng folder
shutil.copytree("important_docs", "backup_important_docs")

Task Scheduling

Sa Windows, gamitin ang Task Scheduler. Sa Linux/Mac, gamitin ang cron. Pwedeng i-schedule ang Python script para tumakbo araw-araw or every hour.
