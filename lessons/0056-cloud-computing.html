Lesson 0056: Cloud Computing Concepts

Pag-unawa sa cloud computing — kung ano ito, ang mga service models (IaaS/PaaS/SaaS), deployment types (public/private/hybrid), at ang mga benepisyo nito.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makapagpapaliwanag ng cloud computing concepts, makapag-differentiate ng IaaS/PaaS/SaaS at public/private/hybrid cloud, at makapagbigay ng mga halimbawa ng cloud services.

Ano ang Cloud Computing?

Ang cloud computing ay ang paghahatid ng computing services — servers, storage, databases, networking, software — sa pamamagitan ng internet. Sa halip na bumili at mag-maintain ng sarili mong physical servers, nagre-rent ka ng resources mula sa cloud providers.

Isipin mo na lang: imbes na magtayo ka ng sarili mong power plant para magamit ang kuryente, kumonekta ka na lang sa electrical grid. Ganun din ang cloud — imbes na bumili ng sariling server, gumamit ka ng computing power on demand.

"Imbes na bibili ka ng sariling server, nagre-rent ka ng computing resources sa cloud (e.g., AWS, Google Cloud, Azure)."

Cloud Service Models: IaaS, PaaS, SaaS

May tatlong pangunahing cloud service models:

  • IaaS (Infrastructure as a Service) — Nagbibigay ng virtualized computing resources tulad ng VMs, storage, at networks. Ikaw ang may kontrol sa OS, apps, at middleware. Halimbawa: AWS EC2, Google Compute Engine.
  • PaaS (Platform as a Service) — Nagbibigay ng platform para mag-develop at mag-deploy ng apps nang hindi na inaalala ang underlying infrastructure. Halimbawa: Google App Engine, Heroku.
  • SaaS (Software as a Service) — Kumpletong software na naka-host sa cloud at ina-access mo sa browser. Halimbawa: Gmail, Google Docs, Microsoft 365.

Cloud Deployment Models

  • Public Cloud — Ang infrastructure ay pag-aari ng third-party provider at shared sa maraming users. Mura at scalable.
  • Private Cloud — Dedicated sa isang organization lang. Mas secure at controlled.
  • Hybrid Cloud — Combination ng public at private cloud. Best of both worlds.

Benefits ng Cloud Computing

  • Scalability — Pwedeng magdagdag o magbawas ng resources on demand.
  • Cost-Efficiency — Pay-as-you-go. Walang upfront cost para sa hardware.
  • Accessibility — Accessible kahit saan, basta may internet.
  • Reliability — Redundancy at backup ng data.

Summary

  • Ang cloud computing ay pag-rent ng computing resources sa internet.
  • Tatlong service models: IaaS (infrastructure), PaaS (platform), SaaS (software).
  • Tatlong deployment models: public, private, hybrid.
  • Benefits: scalability, cost-efficiency, accessibility, reliability.
