Lesson 0002: Parts of a Computer (Hardware Basics)

Pagkilala sa mga bahagi ng computer at ang kani-kanilang function.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakapag-identify ng major parts ng computer at maipaliwanag ang function ng bawat isa.

Input, Output, Processing, at Storage Devices

Ang computer ay may apat na pangunahing kategorya ng hardware:

  • Input Devices — ginagamit para mag-input ng data (keyboard, mouse, microphone, scanner).
  • Output Devices — nagpapakita o naglalabas ng impormasyon (monitor, printer, speaker).
  • Processing Devices — nagpo-process ng data (CPU, GPU).
  • Storage Devices — nag-iimbak ng data (hard drive, SSD, USB flash drive).
"Ang CPU ang utak, RAM ang short-term memory, at ang hard drive ang long-term storage. Kapag mabagal PC mo, baka kulang sa RAM."

Mga Pangunahing Bahagi ng Computer

  • CPU (Central Processing Unit) — Ang utak ng computer. Lahat ng calculations at instructions ay dito dumadaan.
  • RAM (Random Access Memory) — Short-term memory. Dito naka-store ang mga ginagamit mong programs habang gamit mo sila.
  • Hard Drive / SSD — Long-term storage. Dito naka-save ang files, photos, at programs kahit naka-off ang computer.
  • Motherboard — Ang main circuit board na nagdudugtong sa lahat ng parts.
  • PSU (Power Supply Unit) — Nagbibigay ng kuryente sa lahat ng components.

Peripheral Devices

Ang peripherals ay mga devices na nakakabit sa labas ng computer. Kabilang dito ang monitor, keyboard, mouse, printer, speaker, external hard drive, at webcam. Ang ilan ay input (keyboard, mouse), ang iba ay output (monitor, printer), at may ilan na pareho (touchscreen).

Summary

  • Ang hardware ay nahahati sa input, output, processing, at storage devices.
  • CPU ang utak, RAM ang short-term memory, at hard drive/SSD ang long-term storage.
  • Mas mabilis ang SSD kumpara sa HDD.
