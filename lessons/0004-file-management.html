Lesson 0004: File Management & Organization

Paano ayusin ang iyong digital files at folders para laging organisado ang iyong computer.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakagawa ng folders, makakakopya/magmo-move ng files, at makikilala ang mga common file extensions.

Files vs Folders

Ang file ay isang lalagyan ng impormasyon — pwedeng document, picture, video, o program. Ang folder (o directory) naman ay lalagyan ng mga file. Isipin mo na parang filing cabinet: ang folder ay drawer, ang file ay ang papel sa loob.

Common File Extensions

  • .docx — Microsoft Word document
  • .pdf — Portable Document Format
  • .jpg — Image file (JPEG)
  • .exe — Executable program file
  • .zip — Compressed archive file
"Mag-organize ng folders per project or per subject — promise, mas madali maghanap."

Basic File Operations

  • Copy (Ctrl+C) — Gumagawa ng kopya ng file sa clipboard.
  • Cut (Ctrl+X) — Inaalis ang file sa original location para i-paste sa iba.
  • Paste (Ctrl+V) — Inilalagay ang naka-copy o naka-cut na file sa bagong location.
  • Rename (F2) — Pinapalitan ang pangalan ng file.
  • Delete (Delete key) — Sinisilip ang file sa Recycle Bin.
  • Restore — Maaaring i-restore ang file mula sa Recycle Bin kung nagkamali ng delete.

File Explorer Shortcuts

  • Win + E — Buksan ang File Explorer agad.
  • Ctrl + Shift + N — Gumawa ng bagong folder.
  • Alt + P — Ipakita/hide ang preview pane.
  • F2 — Rename selected file.
  • F5 — Refresh ang view.

Mini-Project: Create a Folder Structure

Gumawa ng folder structure sa iyong Documents tulad ng sumusunod: School (na may subfolders per subject), Work (na may subfolders per project), at Personal (na may subfolders na Photos, Music, at Documents). Gamitin ang Ctrl+Shift+N para mabilis gumawa ng bagong folder.

Summary

  • Ang files ay lalagyan ng impormasyon; ang folders ay lalagyan ng mga file.
  • Ang file extension (.docx, .pdf, .jpg) ay nagsasabi kung anong uri ng file ito.
  • Pwedeng i-restore ang deleted file mula sa Recycle Bin.
