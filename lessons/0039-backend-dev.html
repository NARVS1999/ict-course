Lesson 0039: Introduction to Backend Development

Alamin ang server-side scripting — PHP/Node.js, handling requests, sessions, at database connection.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakapag-explain ng backend concepts at makakagawa ng simpleng server-side script na nagdi-display ng dynamic content.

Ano ang Backend Development?

Ang backend ay ang bahagi ng website na nasa server. Hindi ito nakikita ng user pero dito nangyayari ang processing ng data, authentication, at database operations. Ang mga common backend languages ay PHP, Python (Django/Flask), Node.js (Express), at Ruby on Rails.

"Backend ang nagpro-proseso ng data at nag-i-store sa database — hindi ito nakikita ng user."

PHP Basics

<?php
// Variables at echo
$name = "Juan";
echo "Hello, $name!";

// Forms
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $username = $_POST["username"];
    echo "Welcome, $username!";
}

// Sessions
session_start();
$_SESSION["user_id"] = 1;

// MySQL Connection
$conn = new mysqli("localhost", "root", "", "mydb");
$result = $conn->query("SELECT * FROM users");
?>

Node.js with Express

const express = require("express");
const app = express();

// Basic route
app.get("/", (req, res) => {
  res.send("Hello, World!");
});

// Handle form data
app.post("/login", (req, res) => {
  const { username, password } = req.body;
  // Validate and respond
  res.json({ message: "Logged in!" });
});

app.listen(3000, () => {
  console.log("Server running on port 3000");
});

Sessions at Login System

Ang session ay data na naka-store sa server para sa isang specific na user. Ginagamit ito para maalala kung naka-login ang user. Ang isang simpleng login system ay nangangailangan ng: form para sa username/password, validation, session management, at database kung saan naka-store ang user credentials.

Mini-Project: Hello World Backend

Gumawa ng simpleng backend page: sa PHP, gumawa ng index.php na nagpi-print ng "Hello, World!" gamit ang echo. Sa Node.js naman, gumawa ng Express app na may route na nagre-respond ng "Hello, World!".

Summary

  • Ang backend ay server-side code na nagha-handle ng data processing.
  • PHP at Node.js ay common backend languages.
  • Ang sessions ay ginagamit para sa user authentication at state management.
