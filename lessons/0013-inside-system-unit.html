Lesson 0013: Inside the System Unit

Tuklasin ang mga bahagi ng loob ng CPU case — motherboard, RAM, GPU, at iba pa.

Lesson Goal Matutunan ang mga pangunahing internal components ng isang computer at kung paano ligtas na buksan ang system unit.

Pagbubukas ng PC Case nang Ligtas

Bago mo buksan ang system unit, siguraduhing naka-off ang PC at naka-unplug sa saksakan. Mahalaga ito para maiwasan ang electric shock at maprotektahan ang mga components. Gumamit ng anti-static wrist strap kung available, o hawakan muna ang metal na bahagi ng case para ma-discharge ang static electricity. Ang static electricity o ESD (Electrostatic Discharge) ay pwedeng makasira ng sensitive electronics.

"Bago buksan ang PC, patayin at hugutin sa saksakan. Gumamit ng anti-static wrist strap kung meron."

Motherboard — Ang Central ng Lahat

Ang motherboard ang main circuit board ng computer. Dito nakakabit ang lahat ng major components: CPU, RAM, GPU, at storage devices. Makikita mo ang iba't ibang sockets at slots — ang CPU socket, RAM slots (DIMM slots), PCIe x16 slot para sa GPU, at mas maliliit na PCIe slots para sa expansion cards tulad ng Wi-Fi card o sound card.

CPU, RAM, at GPU

Ang CPU (Central Processing Unit) ang utak ng computer — naka-install ito sa CPU socket na may heatsink at fan sa ibabaw. Ang RAM (Random Access Memory) ay temporary memory na ginagamit ng CPU para mag-load ng programs at files. Ang GPU (Graphics Processing Unit) naman ang responsable sa pag-render ng images at video. Ang GPU ay pwedeng integrated (built-in sa CPU) o dedicated (hiwalay na video card sa PCIe slot).

PSU at Power Connectors

Ang PSU (Power Supply Unit) ang nagbibigay ng kuryente sa lahat ng components. May iba't ibang connectors ito: ang 24-pin ATX connector para sa motherboard, 4/8-pin CPU power connector, SATA power para sa storage, at PCIe power para sa GPU. Tiyaking sapat ang wattage ng PSU para sa buong system mo.

Mini-Project: PC Teardown Video

Manood ng YouTube video ng isang PC teardown (halimbawa: "How to Disassemble a Desktop PC"). I-pause ang video at kilalanin ang mga components na nabanggit — motherboard, CPU, RAM, GPU, PSU, at storage. Sumulat ng 5-sentence summary kung ano ang iyong nakita at kung saan matatagpuan ang bawat component.

Summary

  • Laging i-unplug at i-ground ang sarili bago buksan ang PC para maiwasan ang ESD.
  • Ang motherboard ang central circuit board na nagkokonekta sa lahat ng components.
  • Ang CPU ang utak, RAM ang temporary memory, at GPU ang nagha-handle ng graphics.
  • Ang PSU ay may iba't ibang connectors para sa bawat component.
