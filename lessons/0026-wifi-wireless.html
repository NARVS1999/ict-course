Lesson 0026: Wi-Fi & Wireless Networking

Alamin ang Wi-Fi standards, frequencies, security protocols, at kung paano mag-set up ng home wireless network.

Lesson Goal Maunawaan ang mga Wi-Fi standards (802.11a/b/g/n/ac/ax), ang pagkakaiba ng 2.4GHz at 5GHz, at kung paano i-secure ang wireless network.

Wi-Fi Standards (802.11)

Ang Wi-Fi ay base sa IEEE 802.11 standards: 802.11b (11 Mbps, 2.4GHz — luma na), 802.11g (54 Mbps, 2.4GHz), 802.11n (Wi-Fi 4 — hanggang 600 Mbps, pwedeng 2.4 o 5GHz), 802.11ac (Wi-Fi 5 — hanggang 1 Gbps+, 5GHz lang), 802.11ax (Wi-Fi 6 — hanggang 9.6 Gbps, mas efficient sa maraming devices). Ang Wi-Fi 6E ay nagdadagdag ng 6GHz band para sa mas maraming channels at mas kaunting interference.

2.4GHz vs 5GHz Frequencies

Ang 2.4GHz ay mas malayo ang abot at mas mahusay pumasok sa pader, pero mas mabagal at mas madaling ma-interfere (gamit din ito ng microwaves, Bluetooth, at iba pang devices). Ang 5GHz ay mas mabilis at mas kaunti ang interference, pero mas maikli ang range at mas madaling ma-block ng pader. Piliin ang 2.4GHz kung malayo ka sa router; piliin ang 5GHz kung malapit ka at kailangan mo ng bilis.

"Ang 2.4GHz mas malayo abot pero mas mabagal; ang 5GHz mabilis pero maikli range. Pumili ng tamang frequency."

SSID at Security (WPA2/WPA3)

Ang SSID (Service Set Identifier) ay ang pangalan ng Wi-Fi network na nakikita mo. Para sa security, huwag gamitin ang WEP (napakadaling i-crack). Gumamit ng WPA2 (minimum standard) o WPA3 (pinakabago at pinakasecure). Ang WPA3 ay may mas malakas na encryption at mas protektado laban sa password guessing attacks. Iwasan ang paggamit ng default SSID at password ng router.

Channel Interference at Signal Strength

Ang Wi-Fi channels ay pwedeng mag-overlap, lalo na sa 2.4GHz (channels 1, 6, at 11 lang ang hindi nag-o-overlap). Gamitin ang Wi-Fi analyzer apps para makita kung aling channel ang hindi masyadong gamit sa inyong area. Ang signal strength ay sinusukat sa dBm — mas malapit sa 0 (-30 dBm) ang mas malakas, ang -80 dBm ay mahina na.

2. Ano ang pagkakaiba ng 2.4GHz at 5GHz?
3. Ano ang WPA3?

Mini-Project: Check Your Wi-Fi Settings

Pumunta sa Wi-Fi settings ng iyong computer o phone. Hanapin ang iyong kasalukuyang network at tingnan ang mga detalye: SSID (network name), frequency band (2.4GHz o 5GHz), security type (WPA2 o WPA3), at signal strength. Sa Windows, gamitin ang Settings > Network & Internet > Wi-Fi > Properties. I-document ang iyong findings.

Summary

  • Wi-Fi 6 (802.11ax) ang pinakabagong standard — mas mabilis at mas efficient sa maraming devices.
  • 2.4GHz = mas malayo range, mas mabagal; 5GHz = mas mabilis, mas maikli range.
  • Gumamit ng WPA2 o WPA3 para sa security — iwasan ang WEP at default passwords.
  • Sa 2.4GHz, channels 1, 6, at 11 lang ang hindi nag-o-overlap — piliin ang hindi gaanong gamit na channel.
