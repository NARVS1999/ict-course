Lesson 0064: Capstone Project Planning & Execution

Paghahanda para sa capstone project — pagpili ng project, pagpaplano ng scope at timeline, pag-implement gamit ang multiple skills, at paggawa ng documentation.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakapili ng capstone project, makakagawa ng project proposal (scope, tech stack, timeline), at makakapagplano ng documentation.

Pagpili ng Capstone Project

Ang capstone project ay ang pagkakataon mo na i-apply ang lahat ng natutunan mo sa buong kurso. Pumili ng project na nagpapakita ng skills mo sa web development, databases, cybersecurity, at cloud deployment.

Recommended Capstone Projects:

  • IT Asset Management System — Web app para i-track ang assets ng isang kumpanya (computers, monitors, software licenses). May CRUD, search, at user authentication.
  • Network Monitoring Dashboard — Dashboard na nagmo-monitor ng network devices at nagpapakita ng status at alerts.
  • Helpdesk Ticketing System — Web app kung saan pwedeng mag-submit ng tickets ang users at ma-assign sa technicians.
  • Personal Portfolio & Blog — Personal website na may blog functionality, project showcase, at contact form.
  • Cybersecurity Awareness App — Interactive app na nagtuturo ng cybersecurity basics gamit ang quizzes at simulations.
"Pumili ng project na nagpapakita ng skills mo — halimbawa, isang web app na may database, security, at naka-deploy sa cloud."

Project Management Basics

  • Scope — Ano ang kasama at hindi kasama sa project. Dapat realistic at specific.
  • Timeline — Hatiin ang project sa phases: Planning, Development, Testing, Deployment, Documentation.
  • Tools — Gumamit ng project management tools tulad ng Trello, Notion, o simpleng to-do list.

Documentation

Ang documentation ay kasinghalaga ng code. Dapat mayroon kang:

  • README — Overview ng project, tech stack, features, at instructions kung paano i-run.
  • User Manual — Step-by-step guide para sa end-user.
  • Technical Documentation — Architecture, database schema, API endpoints.

Mini-Project: Write a Project Proposal

Pumili ng isang capstone project mula sa list. Sumulat ng project proposal na naglalaman ng: project title, problem statement, scope, tech stack (HTML/CSS/JS, Python, MySQL, cloud platform), at timeline (1 linggo planning, 2 linggo development, 1 linggo testing at deployment).

Summary

  • Pumili ng capstone project na nagpapakita ng iyong natutunang skills.
  • Recommended projects: Asset Management, Network Dashboard, Ticketing System, Portfolio/Blog, Cybersecurity App.
  • Mahalaga ang scope, timeline, at tools sa project management.
  • Documentation (README, user manual, technical docs) ay kasinghalaga ng code.
