Lesson 0015: Memory & Storage

Pag-aralan ang RAM at storage devices — DDR4, DDR5, HDD, SSD, NVMe, at disk management.

Lesson Goal Maunawaan ang pagkakaiba ng RAM at storage, ang mga uri ng RAM at storage devices, at kung paano i-manage ang disks sa Windows.

RAM: Types at Speed

Ang RAM (Random Access Memory) ay temporary memory na ginagamit ng computer para mag-load ng mga programs at data na aktibong ginagamit. Ang mga modernong RAM ay DDR (Double Data Rate): DDR3 (luma), DDR4 (current standard), at DDR5 (pinakabago). Mas mabilis at mas efficient ang bawat bagong henerasyon. Ang bilis ng RAM ay sinusukat sa MHz — halimbawa, DDR4-3200. Para sa mas magandang performance, mag-install ng RAM in pairs (dual-channel) — ilagay sa alternating slots (A2 at B2).

HDD vs SSD: SATA at NVMe

Ang HDD (Hard Disk Drive) ay gumagamit ng spinning magnetic platters — mabagal pero mura at malaki ang capacity. Ang SSD (Solid State Drive) ay gumagamit ng flash memory — sobrang bilis, tahimik, at matibay dahil walang gumagalaw na parts. Ang SSD ay may dalawang uri: SATA SSD (gamit ang SATA cable, hanggang ~550 MB/s) at NVMe M.2 SSD (direkta sa motherboard, hanggang 7000+ MB/s).

"SSD sobrang bilis kumpara sa HDD — laking difference sa boot time. Kung naka-HDD ka pa, upgrade na!"

Disk Management: Partitioning, Formatting, Defrag/TRIM

Sa Windows, gamit ang Disk Management tool, pwedeng mag-create ng partitions (hatiin ang isang drive), mag-format (ihanda ang drive para gamitin), at mag-assign ng drive letters. Para sa HDD, may defragmentation (i-reorganize ang files para mas mabilis basahin). Para sa SSD, huwag mag-defrag — gamitin ang TRIM command (automatic sa Windows) na nagde-delete ng unused data blocks para mapanatili ang bilis.

Quick Quiz

1. Ano ang pagkakaiba ng DDR4 at DDR5?
2. Alin ang mas mabilis: SATA SSD o NVMe SSD?
3. Dapat bang i-defrag ang SSD?

Mini-Project: Check Your System's RAM and Storage

Buksan ang Task Manager (Ctrl+Shift+Esc) at pumunta sa Performance tab. Tingnan ang RAM — ilan ang total at speed (MHz)? Tingnan din ang storage (Disk 0) — HDD ba o SSD? Ilan ang capacity at gaano ang natitira? Bukas din ang System Information (msinfo32) para makita ang detailed RAM info. Isulat ang iyong findings.

Summary

  • RAM ay temporary memory — DDR4 at DDR5 ang common ngayon, mas mabilis ang DDR5.
  • SSD ay mas mabilis kaysa HDD — laking improvement sa boot time at overall performance.
  • NVMe SSD ay mas mabilis kaysa SATA SSD — gamitin kung compatible ang motherboard.
  • Huwag i-defrag ang SSD — gamitin ang TRIM para sa maintenance.
