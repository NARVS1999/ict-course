Lesson 0062: Resume, Portfolio & LinkedIn

Pag-aaral kung paano gumawa ng effective technical resume, online portfolio (GitHub/personal website), at LinkedIn profile para sa IT job applications.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakagawa ng technical resume na nagha-highlight ng skills at projects, makakapag-set up ng GitHub portfolio, at makaka-optimize ng LinkedIn profile.

Technical Resume

Ang technical resume ay iba sa ordinaryong resume. Dapat i-highlight ang iyong technical skills, projects, at certifications. Iwasan ang mahahabang description — maging concise at specific.

Key Sections: Contact Info, Professional Summary, Skills (technical stack), Experience, Projects, Education & Certifications.

"Ang GitHub profile mo ay ang bago mong resume — lagyan ng magagandang projects kahit practice lang."

Online Portfolio

Dalawang uri ng portfolio na mahalaga sa IT:

  • GitHub Profile — Ilagay ang lahat ng iyong code projects. Maganda ang README sa bawat repo na nagpapaliwanag ng project, tech stack, at features.
  • Personal Website — Pwedeng gawin gamit ang HTML/CSS/JS, o gumamit ng platforms tulad ng GitHub Pages, Netlify, o WordPress. Dapat may mga sections: About, Projects, Skills, Contact.

LinkedIn Optimization

  • Headline — Hindi lang "Student" o "Looking for opportunities." Dapat specific: "Aspiring Web Developer | HTML/CSS/JavaScript | IT Student."
  • Summary — 2-3 sentences tungkol sa iyong skills, goals, at passion sa tech.
  • Skills Section — Ilagay ang relevant technical skills at i-ask ang connections na i-endorse ka.
  • Projects — Mag-upload ng screenshots at link sa GitHub repos.

Quick Quiz

1. Ano ang dapat i-highlight ng isang technical resume?
2. Ano ang isang portfolio sa konteksto ng IT?
3. Ano ang dapat ilagay sa LinkedIn headline?

Mini-Project: Create or Update Your LinkedIn Profile

Gumawa ng LinkedIn profile kung wala pa, o i-update ang existing profile. Siguraduhing may professional headline, summary, at hindi bababa sa isang project ang nakalagay. I-share ang link sa profile bilang output.

Summary

  • Technical resume dapat i-highlight ang skills, projects, at certifications.
  • GitHub at personal website ang main portfolio platforms para sa IT.
  • LinkedIn headline dapat specific at nagpapakita ng skills at career goal.
  • Maganda ang README files sa GitHub para ma-explain ang projects.
