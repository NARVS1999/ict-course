Exporting Data to CSV / Excel

Ang CSV (Comma-Separated Values) ay isang simpleng format na nababasa ng Excel, Google Sheets, at ibang tools. Pwedeng mag-export ng data mula sa MySQL gamit ang INTO OUTFILE o mula sa PHP/Python sa pamamagitan ng pag-generate ng CSV file.

-- MySQL export to CSV
SELECT * FROM Students
INTO OUTFILE 'students.csv'
FIELDS TERMINATED BY ','
ENCLOSED BY '"'
LINES TERMINATED BY '\n';

// PHP export
header('Content-Type: text/csv');
header('Content-Disposition: attachment; filename="data.csv"');
$output = fopen('php://output', 'w');
fputcsv($output, ['Name', 'Course', 'Grade']);
fputcsv($output, ['Juan', 'BSIT', 1.5]);
fclose($output);

Basic Analysis gamit ang Python (Pandas)

Ang Pandas ay isang Python library para sa data manipulation at analysis. Gamit ang DataFrame, pwedeng mag-filter, mag-group, at mag-compute ng statistics mula sa datasets.

import pandas as pd

# Basahin ang CSV file
df = pd.read_csv('students.csv')

# Tingnan ang unang 5 rows
print(df.head())

# Basic statistics
print(df.describe())

# Group by course at kunin ang average grade
avg_by_course = df.groupby('course')['grade'].mean()
print(avg_by_course)

Excel Pivot Tables

Ang pivot table sa Excel ay nagbibigay-daan para i-summarize ang malalaking datasets sa pamamagitan ng pag-drag at pag-drop ng fields. Halimbawa: ilagay ang "Course" sa rows, "Grade" sa values (average), at makikita mo ang average grade per course.

Data Visualization

Ang data visualization ay ang pag-represent ng data sa graphical form (charts, graphs, dashboards). Tools: Google Data Studio (free), Looker (enterprise), Tableau Public (free for public data). Ang magandang visualization ay nagpapadali ng pag-unawa sa trends at patterns.

"Hindi lang storage — dapat marunong ka ring mag-analyze ng data para maging useful sa decision-making."
