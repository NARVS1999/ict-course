Lesson 0029: Introduction to Python

Matuto ng Python — ang pinaka-beginner-friendly na programming language. print(), variables, input(), conditionals, at loops.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakapagsulat ng simpleng Python script na may input/output, conditionals, at loops.

Pag-setup ng Python

I-download ang Python mula sa python.org. Pwede kang gumamit ng VS Code, PyCharm, o Thonny bilang editor. Sa VS Code, i-install ang Python extension para may syntax highlighting at auto-complete.

"Python ang pinakamadaling language para sa beginners — parang English magbasa."

Print at Variables

Ang print() ang ginagamit para magpakita ng output. Ang variable naman ay lalagyan ng data.

# Ito ay isang comment
name = "Juan"
age = 18
is_student = True

print("Hello, " + name)
print("Age:", age)
print("Student:", is_student)

User Input

Gamit ang input(), pwedeng mag-enter ang user ng data.

name = input("Enter your name: ")
print("Hello, " + name + "!")

Conditionals (if/elif/else)

age = int(input("Enter your age: "))

if age >= 18:
    print("Adult ka na!")
elif age >= 13:
    print("Teenager ka.")
else:
    print("Bata ka pa.")

Loops (for at while)

Ang for loop ay umuulit sa isang sequence. Ang while loop ay umuulit habang totoo ang condition.

# For loop - mag-print ng 1-5
for i in range(1, 6):
    print(i)

# While loop - magbilang pababa
count = 5
while count > 0:
    print(count)
    count -= 1
print("Blast off!")

Summary

  • Gamitin ang print() para mag-output at input() para sa user input.
