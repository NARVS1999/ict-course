Lesson 0007: Word Processing (Word / Google Docs)

Paano gumawa at mag-format ng dokumento gamit ang Microsoft Word o Google Docs.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakagawa ng maayos na dokumento, makapag-format ng text, makapag-add ng tables at images, at magamit ang track changes.

Creating at Saving Documents

Maaari kang gumawa ng bagong dokumento sa pamamagitan ng pagbukas ng Microsoft Word o Google Docs at pagpili ng "Blank Document" o "New." Sa Word, ginagamit ang Ctrl+S para i-save. Sa Google Docs, automatic nang nase-save sa cloud ang iyong gawa.

Formatting: Fonts, Headings, at Alignment

Pwede mong baguhin ang font style (Arial, Times New Roman, Calibri), font size, at font color. Gamitin ang headings (Heading 1, Heading 2) para maayos na ma-structure ang dokumento. I-align ang text gamit ang Left, Center, Right, o Justify.

"Marunong ka na ba gumawa ng resume na maayos? Dapat naka-consistent font at may tamang margins."

Tables, Images, Headers, at Footers

Ang tables ay ginagamit para mag-organize ng data sa rows at columns. Pwede kang mag-insert ng images mula sa iyong computer. Ang header ay nasa taas ng page (para sa title o logo), ang footer naman ay nasa baba (para sa page number).

Track Changes

Ang Track Changes ay isang feature na nagre-record ng lahat ng edits na ginagawa sa dokumento. Ito ay kapaki-pakinabang kapag may nagrereview ng iyong document — makikita mo kung sino ang nagbago ng ano at kailan.

Mini-Project: Create a Simple Resume

Gumawa ng isang simpleng resume gamit ang Word o Google Docs. Isama ang sumusunod: Name at contact info sa header, Objective, Education, Skills, at Work Experience. Gumamit ng consistent fonts at headings, at siguraduhing maayos ang margins at alignment.

Summary

  • Gumamit ng consistent fonts, headings, at alignment para sa maayos na dokumento.
  • Ang tables at images ay nagpapaganda at nag-o-organize ng content.
  • Track Changes ay kapaki-pakinabang para sa collaboration at review ng dokumento.
