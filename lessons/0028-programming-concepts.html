Lesson 0028: Programming Concepts 101

Alamin ang pangunahing konsepto ng programming — algorithm, flowchart, pseudocode, variables, data types, at control structures.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakapag-explain ng basic programming concepts at makakagawa ng simpleng algorithm gamit ang flowchart at pseudocode.

Algorithm, Flowchart, at Pseudocode

Ang algorithm ay isang step-by-step na paraan para malutas ang isang problema. Para ma-visualize ang algorithm, gumagamit tayo ng flowchart (mga hugis at arrow) o pseudocode (parang code pero English ang gamit).

"Ang programming ay pagbibigay ng sunod-sunod na utos sa computer gamit ang wikang naiintindihan niya."

Variables at Data Types

Ang variable ay parang lalagyan ng data sa memory ng computer. May iba't ibang data types:

  • Integer — buong numero (hal. 5, -3, 100)
  • String — texto o salita (hal. "Hello", "Juan")
  • Boolean — true o false lang

Control Structures

Tatlong uri ng control structures:

  • Sequence — sunod-sunod na execution ng statements
  • Selection — pagpili (if/else) depende sa condition
  • Iteration — pag-uulit (loops) hanggang sa matugunan ang condition

Quick Quiz

1. Ano ang algorithm?
2. Anong data type ang ginagamit para mag-imbak ng text?
3. Anong control structure ang nagpapaulit-ulit ng code?

Mini-Project: Coffee-Making Algorithm

Gumawa ng pseudocode para sa paggawa ng kape. Isulat ang step-by-step process — mula sa paghahanda ng ingredients hanggang sa pag-inom ng kape. Halimbawa: START, Painitin ang tubig, Ilagay ang kape sa tasa, atbp.

Summary

  • Ang algorithm ay step-by-step solution sa problema.
  • Ang flowchart at pseudocode ay ginagamit para i-represent ang algorithm.
  • May tatlong data types: integer, string, boolean.
  • Tatlong control structures: sequence, selection, iteration.
