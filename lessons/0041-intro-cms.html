Lesson 0041: Introduction to CMS (WordPress)

Alamin ang Content Management System at kung paano gumawa ng website gamit ang WordPress nang hindi nangangailangan ng coding.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakapag-install ng WordPress, makakapag-customize ng themes, at makakagawa ng pages at posts.

Ano ang CMS?

Ang CMS (Content Management System) ay software na nagpapahintulot sa users na gumawa, mag-manage, at mag-publish ng content nang hindi nagsusulat ng code. Ang WordPress ang pinakasikat na CMS — ginagamit ng 43% ng websites sa buong mundo.

"WordPress — 43% ng websites sa buong mundo! Kahit hindi marunong mag-code, kayang gumawa ng site."

Pag-install ng WordPress

Para sa local development, gamitin ang XAMPP (Windows) o Local (by Flywheel). I-download ang WordPress mula sa wordpress.org, i-extract sa htdocs folder (sa XAMPP), at i-setup ang database sa phpMyAdmin.

1. I-install ang XAMPP at i-start ang Apache at MySQL
2. I-download ang WordPress at i-extract sa C:\xampp\htdocs\mysite
3. Gumawa ng database sa phpMyAdmin (e.g., "wordpress_db")
4. i-access ang http://localhost/mysite sa browser
5. Sundin ang installation wizard (database name, username: root, password: blank)

Themes, Plugins, at Widgets

  • Themes — Nagkokontrol sa itsura ng website. Pwedeng mag-install ng libre o premium themes.
  • Plugins — Nagdadagdag ng functionality tulad ng contact forms, SEO, security, e-commerce.
  • Widgets — Mga content blocks na pwedeng ilagay sa sidebar, footer, at iba pang widget areas.

Pages vs Posts

  • Pages — Static content tulad ng About, Contact, Homepage. Hindi naka-organize by date.
  • Posts — Blog entries na naka-organize by date at categories. Pwedeng may comments.

Basic Customization

Sa WordPress dashboard, pumunta sa Appearance → Customize para baguhin ang site identity (title, tagline), colors, menus, at homepage settings. Sa Appearance → Menus, pwedeng gumawa ng navigation menu.

3. Ano ang pinagkaiba ng page at post sa WordPress?

Mini-Project: WordPress Local Install

I-install ang XAMPP at WordPress sa local machine. Gumawa ng test post na may title na "My First WordPress Post" at may content tungkol sa iyong natutunan sa araling ito. I-customize ang theme (palitan ang header image o colors).

Summary

  • Ang CMS ay nagpapadali ng website creation nang walang coding.
  • Ang WordPress ay may themes (design) at plugins (functionality).
  • Ang pages ay static, ang posts ay blog entries na naka-organize by date.
