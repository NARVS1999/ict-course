Lesson 0057: Cloud Service Providers

Pagkilala sa mga major cloud providers — AWS, Google Cloud, at Azure — at ang kanilang core services tulad ng compute, storage, at databases.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makapagbibigay ng overview ng AWS, Google Cloud, at Azure services, makapag-explain ng EC2/S3/RDS, at makapag-set up ng free tier account.

Major Cloud Providers

May tatlong major cloud providers sa mundo ngayon:

  • Amazon Web Services (AWS) — Pinakamalaki at pinakamature. Mahigit 200 services ang available. Unang nag-offer ng cloud services noong 2006.
  • Microsoft Azure — Malakas sa enterprise at hybrid cloud. Maganda ang integration sa Microsoft products.
  • Google Cloud Platform (GCP) — Kilala sa data analytics, machine learning, at Kubernetes.
"Kung gusto mong matuto ng cloud, magsimula ka sa AWS at Azure — yan ang hinahanap ng karamihan sa employers ngayon."

Core Services: Compute

  • AWS EC2 (Elastic Compute Cloud) — Virtual server sa cloud. Pwede kang pumili ng OS, CPU, RAM, at storage.
  • Azure Virtual Machines — Parehas sa EC2 pero galing sa Microsoft.
  • Google Compute Engine — VMs sa Google Cloud.

Core Services: Storage

  • AWS S3 (Simple Storage Service) — Object storage para sa files, images, backups, at static website hosting.
  • Azure Blob Storage — Object storage para sa unstructured data.
  • Google Cloud Storage — Unified object storage.

Core Services: Databases

  • AWS RDS (Relational Database Service) — Managed database (MySQL, PostgreSQL, SQL Server, atbp.)
  • Azure SQL Database — Managed SQL Server sa cloud.
  • Google Cloud SQL — Managed MySQL, PostgreSQL, SQL Server.

Free Tier Accounts

Bawat major provider ay may free tier — libreng services na may limitasyon para sa pag-aaral at testing. Halimbawa, ang AWS Free Tier ay may 750 hours ng EC2 t2.micro per month for 12 months, 5GB S3 storage, at marami pang iba. Perfect ito para sa mga nagsisimulang mag-aral ng cloud.

Mini-Project: Sign Up for a Free Tier Account

Pumili ng isang cloud provider (AWS, GCP, o Azure) at mag-sign up para sa kanilang free tier account. Sundin ang registration process, mag-verify ng email at payment method (hindi ka naman sisingilin basta nasa free tier ka lang). Screenshot ang confirmation page at isulat ang experience mo.

Summary

  • Major cloud providers: AWS, Microsoft Azure, Google Cloud.
  • Compute services: EC2 (AWS), VMs (Azure), Compute Engine (GCP).
  • Storage services: S3 (AWS), Blob (Azure), Cloud Storage (GCP).
  • Lahat ng providers ay may free tier para sa pag-aaral.
