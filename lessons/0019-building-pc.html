Lesson 0019: Building a PC from Scratch

Gabay sa pagbuo ng sarili mong computer — mula sa pagpili ng parts hanggang sa first boot at OS installation.

Lesson Goal Matutunan ang buong process ng pag-assemble ng PC, mula sa compatibility checking hanggang sa pag-install ng operating system.

Compatibility Checking

Bago bumili ng parts, siguraduhing compatible ang mga ito. Gamitin ang PC Part Picker (pcpartpicker.com) — awtomatiko nitong chicheck kung tugma ang motherboard sa CPU socket, RAM type, PSU wattage, at case size. Importante ring isaalang-alang ang budget at purpose ng PC (gaming, office, content creation).

Step-by-Step Assembly

1. CPU — I-install sa motherboard socket, i-align ang gold triangle. 2. CPU Cooler — Lagyan ng thermal paste, ikabit ang heatsink at fan. 3. RAM — I-slide sa DIMM slots (A2 at B2 para sa dual-channel). 4. Motherboard sa Case — I-screw ang standoffs, ilagay ang I/O shield, at i-secure ang motherboard. 5. PSU — I-install sa case, ikabit ang 24-pin at CPU power cables. 6. Storage — Ikabit ang SSD/HDD at i-connect ang SATA cable (kung SATA). 7. GPU — I-slide sa PCIe x16 slot at i-secure. 8. Cable Management — Ayusin ang cables para sa magandang airflow.

"Parang Lego ang PC building — masaya pero dapat compatible ang parts. Huwag pwersahin!"

First Boot at OS Installation

Pagkatapos ng assembly, i-on ang PC — dapat makita ang BIOS screen. I-set ang boot order sa USB drive (kung nag-i-install ng OS). I-install ang Windows o Linux sa pamamagitan ng bootable USB. Pagkatapos, i-install ang drivers (chipset, GPU, LAN, audio) at mag-update ng Windows. Huwag kalimutan ang cable management — magandang airflow = mas malamig na components = mas matagal na buhay.

Mini-Project: Build a Budget PC on PC Part Picker

Gamit ang pcpartpicker.com, gumawa ng parts list para sa isang budget PC build (hal. 30,000-40,000 PHP). Pumili ng CPU, motherboard, RAM, storage, PSU, at case. Siguraduhing compatible ang lahat. I-screenshot o i-save ang completed parts list at isulat kung para saan mo gagamitin ang PC na ito (school, gaming, office).

Summary

  • Gumamit ng PC Part Picker para i-verify ang compatibility ng parts bago bumili.
  • Ang tamang sequence ng assembly: CPU → Cooler → RAM → Motherboard → PSU → Storage → GPU.
  • Huwag pwersahin ang parts — kung hindi pumasok, baka mali ang alignment o compatibility.
  • Pagkatapos ng assembly, i-verify sa BIOS na na-detect ang lahat bago mag-install ng OS.
