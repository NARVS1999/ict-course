Lesson 0021: IP Addressing & Subnetting Basics

Unawain ang IPv4, IPv6, subnet masks, at kung paano nagkakaroon ng address ang bawat device sa network.

Lesson Goal Matutunan ang basics ng IP addressing — IPv4 vs IPv6, IP classes, private vs public IP, at subnet masking.

IPv4 vs IPv6

Ang IPv4 ay ang tradisyonal na IP address — 32 bits, 4 na grupo ng numbers (0-255 bawat isa), hal. 192.168.1.1. May limitasyon ito: mga 4.3 bilyong addresses lang — ubos na! Ang IPv6 ay ang kapalit — 128 bits, 8 grupo ng hexadecimal, hal. 2001:0db8:85a3::8a2e:0370:7334. Bilyon-bilyon ang addresses nito — sapat para sa lahat ng devices sa mundo.

IP Classes at Private vs Public

Ang IPv4 ay may mga classes: Class A (1.0.0.0 - 127.255.255.255) para sa malalaking networks, Class B (128.0.0.0 - 191.255.255.255) para sa medium, Class C (192.0.0.0 - 223.255.255.255) para sa maliliit. Ang private IP addresses ay ginagamit sa loob ng local network at hindi naruroute sa internet — ang common ay 192.168.x.x, 10.x.x.x, at 172.16.x.x. Ang public IP naman ay unique sa buong mundo at kailangan para makakonekta sa internet.

Subnet Mask at CIDR

Ang subnet mask ay nagsasabi kung aling parte ng IP address ang network at alin ang host. Halimbawa, ang 255.255.255.0 ay nangangahulugang 24 bits para sa network at 8 bits para sa hosts — kaya 254 usable devices. Ang CIDR notation ay pinaikli: /24 = 255.255.255.0. Kung /16 = 255.255.0.0 (maraming hosts), kung /30 = 255.255.255.252 (2 usable hosts lang — para sa point-to-point).

"Ang subnet mask nagsasabi kung ilan ang pwedeng devices sa isang network. 255.255.255.0 means 254 usable hosts."

Mini-Project: Check Your IP Configuration

Buksan ang Command Prompt (cmd) at i-type ang ipconfig (Windows) o ifconfig (macOS/Linux). Hanapin ang IPv4 address, subnet mask, at default gateway ng iyong computer. Isulat ang mga ito at tukuyin kung anong IP class ito at kung ilan ang pwedeng hosts base sa subnet mask.

Summary

  • IPv4 ay 32 bits (4.3B addresses), IPv6 ay 128 bits (halos unlimited).
  • Private IPs (192.168.x.x, 10.x.x.x) ay para sa local network; public IP ay para sa internet.
  • Subnet mask / CIDR ang nagsasabi kung ilang devices ang pwedeng magkasya sa network.
  • 255.255.255.0 (/24) = 254 usable hosts — pinaka-common sa bahay at opisina.
