Lesson 0044: SQL Basics

Pag-aaral ng SQL — ang wika ng database para mag-query, mag-insert, mag-update, at mag-delete ng data.

Lesson Goal Matutunan ang basic SQL commands: SELECT, INSERT, UPDATE, DELETE, at mga filtering techniques.

SELECT — Pagkuha ng Data

Ang SELECT ay ginagamit para kumuha ng data mula sa isang table. Pwedeng piliin ang lahat ng columns (*) o specific columns lang. Gamit ang WHERE, pwedeng i-filter ang results. Ang ORDER BY ay para sa pag-aayos, at LIMIT para sa bilang ng rows na ibabalik.

-- Kunin lahat ng records
SELECT * FROM Students;

-- Kunin specific columns na may filter
SELECT first_name, last_name, email
FROM Students
WHERE course = 'BSIT'
ORDER BY last_name ASC
LIMIT 10;

INSERT, UPDATE, DELETE

Ang INSERT ay nagdaragdag ng bagong record, ang UPDATE ay nagbabago ng existing records, at ang DELETE ay nagtatanggal ng records. Laging mag-ingat sa UPDATE at DELETE — gamitin ang WHERE para hindi maapektuhan ang lahat ng records.

-- Magdagdag ng bagong estudyante
INSERT INTO Students (student_id, first_name, last_name, email, course)
VALUES (101, 'Juan', 'Dela Cruz', 'jdcruz@email.com', 'BSIT');

-- Mag-update ng email
UPDATE Students
SET email = 'jdcruz2026@email.com'
WHERE student_id = 101;

-- Mag-delete ng record
DELETE FROM Students
WHERE student_id = 101;

Filtering: LIKE, BETWEEN, IN

Gamitin ang LIKE para sa pattern matching (% ay wildcard), BETWEEN para sa range, at IN para sa list ng values.

-- LIKE: maghanap ng pangalan na naglalaman ng "Maria"
SELECT * FROM Students WHERE first_name LIKE '%Maria%';

-- BETWEEN: mga estudyanteng nag-enroll sa pagitan ng dalawang petsa
SELECT * FROM Enrollments
WHERE enrollment_date BETWEEN '2026-01-01' AND '2026-06-30';

-- IN: mga course na specific lang
SELECT * FROM Students WHERE course IN ('BSIT', 'BSCS', 'BSEE');

Aggregate Functions

Ang aggregate functions ay nagko-compute ng iisang value mula sa maraming rows: COUNT (bilang), SUM (kabuuan), AVG (average), MIN (pinakamaliit), at MAX (pinakamalaki).

SELECT
    COUNT(*) AS total_students,
    AVG(age) AS average_age,
    MAX(gpa) AS highest_gpa,
    MIN(gpa) AS lowest_gpa
FROM Students
WHERE course = 'BSIT';

Summary

  • SELECT ay para kumuha ng data; INSERT para magdagdag; UPDATE para magbago; DELETE para magtanggal.
  • Gamitin ang LIKE, BETWEEN, at IN para sa filtering.
  • Aggregate functions: COUNT, SUM, AVG, MIN, MAX para sa computations.
