Lesson 0017: BIOS/UEFI & Boot Process

Alamin ang BIOS at UEFI, POST, boot order, at kung paano i-configure ang basic system settings.

Lesson Goal Maunawaan ang boot process ng computer at kung paano ma-access at gamitin ang BIOS/UEFI setup.

BIOS vs UEFI

Ang BIOS (Basic Input/Output System) ay ang firmware na nag-b-boot ng computer — ito ang unang tumatakbo pagpindot mo ng power button. Ang mas bagong version ay UEFI (Unified Extensible Firmware Interface) — mas moderno, may graphical interface, sumusuporta sa mas malalaking drives (2TB+), at mas mabilis mag-boot. Karamihan sa mga computer ngayon ay UEFI na, pero madalas pa rin tawaging BIOS.

Pag-access ng BIOS at Mga Setting

Para pumasok sa BIOS/UEFI, pindutin ang DEL, F2, o F10 kaagad pagkatapos i-on ang computer (depende sa manufacturer). Sa loob ng BIOS makikita mo ang: Boot Order (alin ang priority — USB, SSD/HDD, DVD), Virtualization (enable para sa virtual machines), Secure Boot (security feature), at XMP/DOCP (para sa RAM speed).

"Kung ayaw mag-boot, pumasok sa BIOS at icheck kung nade-detect ang hard drive."

POST at Beep Codes

Ang POST (Power-On Self-Test) ay ang initial diagnostic test na ginagawa ng computer pag-on. Kung may problema, maririnig ang beep codes — halimbawa, 1 long beep + 2 short beeps ay usually GPU issue. Kung walang beep at walang display, posibleng may problem sa RAM o CPU. Ang CMOS battery (CR2032) ay nagpapanatili ng BIOS settings kahit naka-off ang PC — kung patay na ito, mawawala ang oras at settings sa tuwing mag-bboot.

Mini-Project: Explore Your BIOS/UEFI

Kung kaya mong i-access ang BIOS/UEFI ng computer mo, pumasok at hanapin ang boot order setting — i-take note kung ano ang nakalagay bilang first boot device. Kung bawal o hindi ka sigurado, manood ng YouTube video na nagpapakita ng BIOS setup ng kahit anong motherboard (hal. "How to enter BIOS and change boot order"). I-drawing o i-screenshot ang boot order screen at isulat kung ano ang mga options.

Summary

  • BIOS/UEFI ang unang software na tumatakbo pag-on ng computer — UEFI ang modern version.
  • Pindutin ang DEL, F2, o F10 para pumasok sa BIOS setup.
  • Ang POST ay nagche-check kung maayos ang hardware — ang beep codes ay nag-i-indicate ng specific problems.
  • Ang CMOS battery ay nagpapanatili ng BIOS settings kahit naka-off ang PC.
