Lesson 0018: Common PC Problems & Troubleshooting

Matutunan kung paano i-diagnose at i-ayos ang mga common computer problems — no power, BSOD, slow performance, at higit pa.

Lesson Goal Makapag-troubleshoot ng common PC issues gamit ang built-in tools ng Windows.

No Power, No Display, at Beeping

Kung walang power: suriin ang power cable, PSU switch, at outlet. Kung may power pero walang display: tingnan kung naka-connect ang monitor cable sa GPU (hindi sa motherboard kung may dedicated GPU). Kung may beeping: i-search ang beep pattern online — 1 long + 2 short ay GPU, continuous beeps ay RAM issue. Subukan mong i-reseat ang RAM (tanggalin at ibalik) at GPU.

BSOD (Blue Screen of Death)

Ang BSOD ay nangyayari kapag nagka-critical error ang Windows. Maganda ang BSOD dahil may stop code (hal. MEMORY_MANAGEMENT, VIDEO_TDR_FAILURE, CRITICAL_PROCESS_DIED) na nagsasabi kung saan ang problema. I-search ang stop code online para sa specific solution. Pwedeng driver issue, faulty RAM, overheating, o corrupted system files.

Slow Performance at Task Manager

Gamit ang Task Manager (Ctrl+Shift+Esc), makikita mo kung aling programs ang gumagamit ng CPU, RAM, Disk, at Network. Kung 100% ang disk usage, baka may program na stuck o HDD ang gamit mo (upgrade to SSD!). Pwede ring magkaroon ng malware o masyadong maraming startup programs. I-right-click ang task at piliin ang "End Task" para isara ang resource-heavy programs.

"Bago tawagin ang technician, subukan muna mag-restart — maraming problema nasosolve ng restart."

Safe Mode at System Restore

Ang Safe Mode ay nag-bboot ang Windows na may minimal drivers lang — mainam ito para mag-uninstall ng problematic drivers o programs. Pumasok sa Safe Mode sa pamamagitan ng pag-press ng F8 (sa lumang Windows) o pag-restart habang hawak ang Shift. Ang System Restore naman ay nagbabalik ng Windows sa isang previous state (restore point) — parang time machine para sa system files.

Windows Recovery

Kung hindi mag-boot ang Windows, ang Windows Recovery Environment (WinRE) ay may options: Startup Repair (awtomatikong ayusin), Command Prompt (advanced troubleshooting), at Reset This PC (ire-install ang Windows pero pwedeng keep files).

Quick Quiz

1. Ano ang ibig sabihin ng BSOD?
2. Anong tool ang ginagamit para makita ang CPU at RAM usage?
3. Para saan ang Safe Mode?

Mini-Project: Monitor Your PC Performance

Buksan ang Task Manager (Ctrl+Shift+Esc) at pumunta sa Performance tab. I-observe ang CPU, Memory, at Disk usage. Kung may program na gumagamit ng maraming resources, i-click ang Processes tab, i-sort by CPU usage, at i-note ang top 5 processes. Subukan ding buksan ang Startup Apps tab at tingnan kung aling apps ang naka-enable. I-document ang iyong observation.

Summary

  • I-restart muna bago mag-troubleshoot — maraming issues nasosolve nito.
  • Ang BSOD ay may stop code na nagsasabi kung ano ang problema.
  • Task Manager ang ginagamit para i-monitor ang performance at mag-end ng problematic processes.
  • Safe Mode at System Restore ay powerful tools para sa pag-aayos ng Windows.
