Lesson 0060: AI, Machine Learning & Automation

Pag-unawa sa Artificial Intelligence (AI), Machine Learning (ML), Deep Learning, at kung paano ginagamit ang mga ito sa real-world applications at automation.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makapag-differentiate ng AI/ML/Deep Learning, makapagbigay ng examples ng AI applications, at makagamit ng no-code AI tool tulad ng Teachable Machine.

AI vs Machine Learning vs Deep Learning

  • Artificial Intelligence (AI) — Malawak na field na naglalayong gumawa ng machines na kayang mag-isip at matuto tulad ng tao. Kasama dito ang reasoning, problem-solving, at language understanding.
  • Machine Learning (ML) — Subset ng AI. Dito tinuturuan ang computer gamit ang data — hindi naka-program ng explicit rules, kundi natututo ito sa patterns.
  • Deep Learning (DL) — Subset ng ML na gumagamit ng neural networks na maraming layers. Ito ang ginagamit sa image recognition, natural language processing, at self-driving cars.
"Hindi mo kailangang maging data scientist para gumamit ng AI — maraming APIs na pwedeng i-integrate."

Applications ng AI

  • Chatbots — ChatGPT, Google Bard, customer service bots.
  • Image Recognition — Pag-identify ng objects sa pictures (Google Photos, Facebook auto-tag).
  • Recommendation Systems — Netflix suggestions, YouTube recommendations, Shopee/Lazada product suggestions.
  • Speech Recognition — Siri, Google Assistant, Alexa.

No-Code / Low-Code AI Tools

Pwedeng matuto ng AI kahit hindi marunong mag-code. Ilan sa mga tools:

  • Google Teachable Machine — Sanayin ang computer na mag-recognize ng images, sounds, o poses gamit ang webcam.
  • ChatGPT API — Integrate ang GPT sa iyong app para magkaroon ng natural language capabilities.
  • Runway ML — Drag-and-drop machine learning models.

Robotic Process Automation (RPA)

Ang RPA ay paggamit ng software robots para i-automate ang repetitive tasks — tulad ng data entry, file processing, at email sending. Tools: UiPath, Automation Anywhere, Power Automate. Layunin nito: makatipid ng oras at mabawasan ang human error.

Mini-Project: Train an Image Classifier

Pumunta sa teachablemachine.withgoogle.com at gumawa ng image classifier. Mag-train ng model na mag-recognize ng dalawang klase ng objects (halimbawa: "ballpen" vs "cellphone"). Gamitin ang webcam para kumuha ng sample images. I-test ang model at i-screenshot ang result.

Summary

  • AI ang malawak na field; ML ay subset na natututo mula sa data; Deep Learning ay subset ng ML gamit neural networks.
  • Maraming applications: chatbots, image recognition, recommendation systems, RPA.
  • May no-code tools tulad ng Teachable Machine para matuto ng AI nang walang coding.
  • RPA ay pag-automate ng repetitive tasks gamit software robots.
