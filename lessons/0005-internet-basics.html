Lesson 0005: Internet Basics & Web Browsers

Paano gumagana ang internet, ang papel ng ISP at DNS, at paano gamitin ang web browser nang epektibo.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakapagpaliwanag kung paano gumagana ang internet, ISP, DNS, servers, at web browsers.

ISP at DNS

Ang ISP (Internet Service Provider) ay ang kumpanyang nagbibigay sa iyo ng internet connection — halimbawa ay PLDT, Globe, o Converge. Sila ang nagkokonekta ng iyong computer sa malawak na network ng internet.

Ang DNS (Domain Name System) naman ay parang telephone book ng internet. Kapag nag-type ka ng "google.com," tinatanong ng browser mo ang DNS kung anong IP address ang katumbas ng domain name na iyon (hal. 142.250.80.46). Kapag nakuha na ang IP address, doon na mapupunta ang request mo sa server.

"Kapag nag-type ka ng google.com, nagtatanong muna sa DNS 'Anong address nito?' bago pumunta sa server."

Servers at Web Browsers

Ang server ay isang computer na naka-imbak ng mga websites at naghahatid ng content kapag may nag-request. Ang web browser naman ay ang program na ginagamit mo para ma-access ang web — Chrome, Firefox, Edge, Safari, atbp.

URLs, Domains, at Hyperlinks

Ang URL (Uniform Resource Locator) ay ang address ng isang webpage (hal. https://www.google.com). Ang domain ay ang pangalan ng website (google.com). Ang hyperlinks ay mga clickable text o image na nagdadala sa iyo sa ibang page.

Mini-Project: Inspect a Website with DevTools

Pumunta sa anumang website at pindutin ang F12 (o right-click > Inspect). Pumunta sa Network tab at i-refresh ang page. Makikita mo ang lahat ng requests na ginagawa ng browser — images, scripts, at iba pa. I-screenshot o isulat ang tatlong uri ng requests na makikita mo.

Summary

  • ISP ang nagbibigay ng internet connection; DNS ang nagta-translate ng domain names to IP addresses.
  • Ang server ay nag-iimbak ng websites; ang browser ang ginagamit para ma-access ang mga ito.
  • URL ang address ng webpage; domain ang pangalan ng website; hyperlinks ang nagkokonekta ng pages.
