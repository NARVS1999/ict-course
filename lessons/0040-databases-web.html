Lesson 0040: Databases for the Web

Matuto ng databases — CRUD operations, SQL, foreign keys, at paano gamitin ang database sa web applications.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakagawa ng MySQL database at tables, at makakapagsagawa ng basic CRUD operations gamit ang SQL queries.

Ano ang Database?

Ang database ay structured na koleksyon ng data. Sa web, ginagamit ang databases para mag-imbak ng users, posts, products, atbp. Ang MySQL ay isa sa pinakasikat na database management system.

"Ang database parang Excel file na sobrang laki at mabilis maghanap — dito naka-store ang users, posts, products."

Creating Databases and Tables

Gamit ang phpMyAdmin o MySQL CLI, pwedeng gumawa ng database at tables.

-- Gumawa ng database
CREATE DATABASE mywebsite;

-- Pumili ng database
USE mywebsite;

-- Gumawa ng table
CREATE TABLE users (
  id INT AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(100) NOT NULL,
  email VARCHAR(100) UNIQUE NOT NULL,
  age INT,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CRUD Operations

Ang CRUD ay Create, Read, Update, Delete — ang apat na basic operations sa database.

-- CREATE (Insert)
INSERT INTO users (name, email, age)
VALUES ('Juan', 'juan@email.com', 20);

-- READ (Select)
SELECT * FROM users;
SELECT name, email FROM users WHERE age > 18;
SELECT * FROM users ORDER BY name ASC;

-- UPDATE
UPDATE users
SET age = 21
WHERE name = 'Juan';

-- DELETE
DELETE FROM users WHERE id = 1;

Relationships: One-to-Many

Ang foreign key ay nagli-link ng isang table sa isa pa. Halimbawa: ang isang user ay pwedeng magkaroon ng maraming posts.

CREATE TABLE posts (
  id INT AUTO_INCREMENT PRIMARY KEY,
  title VARCHAR(200) NOT NULL,
  content TEXT,
  user_id INT,
  FOREIGN KEY (user_id) REFERENCES users(id)
);

-- Join query
SELECT users.name, posts.title
FROM users
JOIN posts ON users.id = posts.user_id;

Gamit ang PHP PDO

<?php
$pdo = new PDO("mysql:host=localhost;dbname=mywebsite",
               "root", "");

$stmt = $pdo->prepare("SELECT * FROM users WHERE id = ?");
$stmt->execute([1]);
$user = $stmt->fetch();

echo $user["name"];
?>

Summary

  • Ang CRUD ay Create, Read, Update, Delete — basic database operations.
  • Ang foreign key ay nagli-link ng tables para sa relationships.
  • Ang SELECT ay ginagamit para kumuha ng data mula sa database.
