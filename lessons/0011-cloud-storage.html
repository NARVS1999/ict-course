Lesson 0011: Cloud Storage & Collaboration

Paano gamitin ang cloud storage para mag-save, mag-share, at makipag-collaborate online.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakapag-upload at makakapag-share ng files sa cloud, makakagamit ng real-time collaboration, at mauunawaan ang version history.

Ano ang Cloud Storage?

Ang cloud storage ay serbisyo kung saan naka-store ang iyong files sa malalayong servers sa internet, hindi sa iyong local computer. Ibig sabihin, maa-access mo ang iyong files kahit saan — sa phone, laptop, o tablet — basta may internet connection. Sikat na providers: Google Drive, OneDrive, at Dropbox.

Uploading, Sharing, at Permissions

Pwedeng mag-upload ng files sa cloud sa pamamagitan ng drag-and-drop o "New" button. Kapag mag-share, pwede mong i-set ang permissions — "Viewer" (tingin lang), "Commenter" (pwedeng mag-comment), o "Editor" (pwedeng mag-edit). Pwede ring mag-set ng expiration date o password para sa shared link.

"Imbes na mag-email ng attachment pabalik-balik, gumamit ng shared Google Docs — iisang file, lahat nag-e-edit."

Real-Time Collaboration

Ang isa sa pinakamagandang feature ng cloud apps ay ang real-time collaboration — maraming tao ang pwedeng mag-edit ng iisang document nang sabay-sabay. Makikita mo kung sino ang nag-e-edit at kung anong bahagi ang binabago nila (may colored cursor bawat user).

Version History

Ang version history ay nagse-save ng lahat ng mga pagbabagong ginawa sa isang file. Pwede mong tingnan ang mga naunang version at i-restore ang lumang version kung kinakailangan. Sa Google Docs, ito ay nasa File > Version History > See Version History.

Mini-Project: Collaborate on a Shared Google Doc

Gumawa ng Google Doc at i-share ito sa isang kaklase o kaibigan na may "Editor" permission. Magsulat kayo nang sabay-sabay ng isang listahan ng inyong mga paboritong websites. Pagkatapos, tingnan ang Version History (File > Version History) para makita kung sino ang nag-edit ng ano.

Summary

  • Ang cloud storage ay nagpapahintulot na ma-access ang files kahit saan.
  • Pwede kang mag-share ng files na may specific permissions: Viewer, Commenter, o Editor.
  • Ang real-time collaboration at version history ay mahalagang features ng cloud apps.
