Lesson 0037: CSS3 & Responsive Design

Matuto ng CSS para istilo ang web pages — selectors, box model, Flexbox, Grid, at media queries.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakapag-apply ng CSS styles sa HTML at makakagawa ng responsive layout gamit ang Flexbox at media queries.

CSS Selectors at Properties

Ang CSS (Cascading Style Sheets) ang nagpapaganda ng website. Pwedeng mag-style ng elements gamit ang selectors.

"CSS ang nagpapaganda ng website. Gamitin ang Flexbox para hindi na mahirapan sa pag-a-align."
/* Element selector */
h1 { color: blue; font-size: 32px; }

/* Class selector */
.card { background: white; border-radius: 8px; }

/* ID selector */
#header { background: #333; color: white; }

/* Pseudo-class */
a:hover { text-decoration: underline; }

Box Model

Ang bawat HTML element ay may box na binubuo ng: content, padding (space sa loob), border, at margin (space sa labas).

div {
  width: 300px;
  padding: 20px;
  border: 2px solid black;
  margin: 10px;
}

Flexbox

Ang display: flex ay nag-a-align ng items nang pahalang o patayo. Mas madali kaysa sa traditional float method.

.container {
  display: flex;
  justify-content: center;   /* Horizontal alignment */
  align-items: center;       /* Vertical alignment */
  gap: 10px;                 /* Space between items */
}

.item { flex: 1; }           /* Equal width items */

CSS Grid

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}

Media Queries at Responsive Design

Gamit ang media queries, pwedeng mag-apply ng ibang styles depende sa screen size.

/* Default: desktop styles */
.container { display: flex; }

/* Tablet */
@media (max-width: 768px) {
  .container { flex-direction: column; }
}

/* Mobile */
@media (max-width: 480px) {
  body { font-size: 14px; }
}

Colors at Typography

body {
  font-family: 'Segoe UI', sans-serif;
  line-height: 1.6;
  color: #333;
  background-color: #f9f9f9;
}

  • Ang CSS ay ginagamit para mag-style ng HTML elements.
