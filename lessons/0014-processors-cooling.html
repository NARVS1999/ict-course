Lesson 0014: Processors & Cooling

Alamin ang tungkol sa CPU — ang utak ng computer — at kung paano ito pinapalamig.

Lesson Goal Maunawaan ang pagkakaiba ng Intel at AMD processors, ang konsepto ng cores/threads/clock speed, at kung paano mag-install ng CPU cooler.

CPU Brands: Intel vs AMD

Dalawang major na brand ng processors sa merkado: Intel at AMD. Ang Intel ay may Core series (i3, i5, i7, i9) habang ang AMD ay may Ryzen series (3, 5, 7, 9). Pareho silang magaling — ang Intel ay kilala sa single-core performance at gaming, samantalang ang AMD ay nag-aalok ng mas maraming cores at threads sa mas murang presyo. Ang socket ng processor ay dapat tugma sa motherboard — halimbawa, Intel LGA1700 o AMD AM5.

Cores, Threads, at Clock Speed

Ang core ay parang isang indibidwal na utak sa loob ng CPU — mas maraming cores, mas maraming tasks ang kayang sabayang hawakan. Ang thread ay virtual core na tumutulong sa multitasking. Ang clock speed (sinusukat sa GHz) ay ang bilis ng bawat core — mas mataas na GHz, mas mabilis ang pag-process. Halimbawa, ang 3.5 GHz ay nangangahulugang 3.5 bilyong cycles per second.

Heatsink, Fan, at Thermal Paste

Ang CPU ay umiinit kapag tumatakbo, kaya kailangan ng heatsink (metal na nag-aabsorb ng init) at fan (na nagpapalabas ng init). Sa pagitan ng CPU at heatsink, nilalagyan ng thermal paste para mas epektibo ang paglipat ng init. Kung tuyo na ang thermal paste, pwedeng mag-overheat ang CPU. Mayroon ding liquid cooling para sa high-performance PCs — gumagamit ito ng liquid para mas mabilis na mag-dissipate ng init.

"Kung nag-o-overheat ang PC, baka tuyo na ang thermal paste — palitan every 2-3 years."

Quick Quiz

1. Ano ang dalawang major CPU brands?
2. Ano ang sinusukat ng clock speed (GHz)?
3. Ano ang gamit ng thermal paste?

Mini-Project: CPU Comparison

Magsaliksik ng Intel Core i5 (latest generation) at AMD Ryzen 5 (latest generation). I-compare ang kanilang specs: number of cores, threads, base clock speed, boost clock speed, TDP (power consumption), at presyo. Gumawa ng table sa notebook o document na nagpapakita ng pagkakaiba at isulat kung alin ang mas maganda para sa iyong pangangailangan.

Summary

  • Intel at AMD ang dalawang pangunahing CPU brands — pumili base sa budget at pangangailangan.
  • Mas maraming cores at threads = mas magaling sa multitasking.
  • Ang clock speed (GHz) ay nagsasabi kung gaano kabilis ang CPU.
  • Ang thermal paste ay mahalaga para sa tamang pagpapalamig ng CPU — palitan tuwing 2-3 taon.
