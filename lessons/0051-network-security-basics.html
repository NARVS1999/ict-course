Lesson 0051: Network Security Basics

Pag-unawa sa firewalls, IDS/IPS, Wi-Fi security, at VPN para sa ligtas na network.

Lesson Goal Matutunan ang mga pangunahing konsepto ng network security at kung paano protektahan ang iyong network.

Firewalls

Ang firewall ay isang system na sumasala ng incoming at outgoing network traffic batay sa security rules. Ang Windows Firewall ay built-in sa Windows — pwedeng mag-configure ng allowed/blocked applications at ports. Ang iptables naman ay firewall para sa Linux.

# Windows Firewall: Check status
netsh advfirewall show allprofiles

# Linux iptables: Allow SSH
iptables -A INPUT -p tcp --dport 22 -j ACCEPT

# Block all incoming traffic except established connections
iptables -A INPUT -m state --state ESTABLISHED,RELATED -j ACCEPT
iptables -P INPUT DROP

IDS at IPS

Ang IDS (Intrusion Detection System) ay nagmo-monitor ng network traffic at nag-aalert kapag may suspicious activity. Ang IPS (Intrusion Prevention System) naman ay aktibong pumipigil sa detection — hinaharangan nito ang malicious traffic bago pa man umabot sa system.

Wi-Fi Security

Ang pinakamahusay na Wi-Fi security ay WPA3 (WPA2 kung hindi supported). I-disable ang WPS (Wi-Fi Protected Setup) dahil vulnerable ito sa brute force attacks. Gumamit ng malakas na Wi-Fi password at i-enable ang guest network para sa mga bisita para hindi nila ma-access ang main network.

VPN

Ang VPN (Virtual Private Network) ay nag-e-encrypt ng iyong internet connection at nagre-route nito sa isang remote server. Gamitin ang VPN kapag: gumagamit ng public Wi-Fi (cafe, airport), gusto itago ang IP address, o mag-access ng geo-restricted content. Hindi perpekto ang VPN — pumili ng trusted provider na walang logs policy.

"Ang firewall ay gatekeeper — sinasala nito kung anong traffic papasok o lalabas ng network."

Mini-Project: Check Windows Firewall

Buksan ang Windows Defender Firewall sa Control Panel. I-check kung anong mga inbound at outbound rules ang kasalukuyang active. Tingnan kung ang "File and Printer Sharing" ay naka-block o naka-allow. I-screenshot ang firewall status bilang ebidensya.

Summary

  • Firewall = sumasala ng traffic; IDS = detection; IPS = prevention.
  • WPA3 ang pinakasecure na Wi-Fi; i-disable ang WPS.
  • VPN ay nag-e-encrypt ng connection — gamitin sa public Wi-Fi at para sa privacy.
