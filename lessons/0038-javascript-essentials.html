Lesson 0038: JavaScript Essentials

Matuto ng JavaScript — variables, DOM manipulation, events, at Fetch API para sa interactive web pages.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakapagdagdag ng interactivity sa HTML page gamit ang JavaScript events at DOM manipulation.

Variables, Data Types, at Functions Recap

Ang JavaScript ay ang programming language ng web. Pwedeng gumamit ng var, let, o const para sa variables.

"JavaScript ang nagbibigay buhay sa website — kapag may click, may animation, may pop-up."
// Variables at Data Types
let name = "Juan";
const age = 20;
let isStudent = true;
let scores = [85, 90, 78];
let person = { firstName: "Maria", lastName: "Santos" };

// Functions
function greet(name) {
  return "Hello, " + name + "!";
}

console.log(greet("Juan"));

DOM Manipulation

Ang DOM (Document Object Model) ay ang representation ng HTML page sa JavaScript. Pwedeng magbasa o magbago ng elements.

// Select elements
let heading = document.getElementById("title");
let paragraphs = document.querySelectorAll("p");
let button = document.querySelector(".btn");

// Change content
heading.textContent = "New Title!";
heading.innerHTML = "<em>Styled Title</em>";

// Change styles
heading.style.color = "red";
heading.classList.add("highlight");

Events

Ang events ay mga aksyon na nagti-trigger ng JavaScript code. Kasama na rito ang click, submit, keypress, mouseover, atbp.

// Click event
button.addEventListener("click", function() {
  alert("Button clicked!");
});

// Form submit event
form.addEventListener("submit", function(e) {
  e.preventDefault();  // Pigilan ang page reload
  console.log("Form submitted!");
});

// Keypress event
input.addEventListener("keyup", function(e) {
  console.log("Key pressed:", e.key);
});

Fetch API

Ang Fetch API ay ginagamit para mag-send ng HTTP requests mula sa browser. Pwedeng kumuha ng data mula sa server nang hindi nagre-reload ng page.

// GET request
fetch("https://api.example.com/users")
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error("Error:", error));

// POST request
fetch("https://api.example.com/users", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ name: "Juan", age: 20 })
});
