Lesson 0008: Spreadsheets (Excel / Google Sheets)

Paano gumamit ng spreadsheets para sa data organization, formulas, at basic analysis.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakagawa ng spreadsheet, makagamit ng basic formulas (SUM, AVERAGE, IF), at makapag-sort at mag-filter ng data.

Cells, Rows, at Columns

Ang spreadsheet ay binubuo ng cells (kahon), rows (pahalang — number 1, 2, 3), at columns (patayo — A, B, C). Ang bawat cell ay may address tulad ng A1, B2, C3. Dito mo ilalagay ang iyong data — numbers, text, o formulas.

Basic Formulas: SUM, AVERAGE, IF, VLOOKUP

  • =SUM(A1:A10) — Nag-aadd ng lahat ng numbers sa range na A1 hanggang A10.
  • =AVERAGE(A1:A10) — Kinukuha ang average ng numbers.
  • =IF(A1>50,"Pass","Fail") — Conditional logic. Kung ang A1 ay > 50, magpapakita ng "Pass", kung hindi, "Fail".
  • =VLOOKUP — Naghahanap ng value sa isang table at nagre-return ng katumbas na value mula sa ibang column.
"Excel ang pinakaimportante sa data entry at analysis. Practice lang nang practice ng formulas."

Sorting, Filtering, at Conditional Formatting

Pwede mong sort ang data pataas (A-Z) o pababa (Z-A). Ang filter naman ay nagpapakita lang ng data na gusto mong makita. Ang conditional formatting ay nagha-highlight ng cells batay sa kanilang value — halimbawa, i-highlight ng pula ang mga benta na mababa sa 100.

Charts at Pivot Tables (Intro)

Pwede kang gumawa ng charts (bar, line, pie) para mas madaling ma-visualize ang data. Ang pivot table ay isang advanced na tool para mag-summarize ng malaking data nang mabilis.

Mini-Project: Create a Budget Spreadsheet

Gumawa ng budget spreadsheet sa Excel o Google Sheets. Gumawa ng listahan ng iyong monthly expenses (renta, pagkain, internet, load, atbp.). Gamitin ang SUM para sa total at AVERAGE para sa average ng expenses. Lagyan ng conditional formatting — i-highlight ng pula ang expenses na lampas sa 1000.

Summary

  • Ang spreadsheet ay binubuo ng cells, rows, at columns.
  • Ang SUM at AVERAGE ay basic formulas; ang IF at VLOOKUP ay mas advanced.
  • Ang sorting, filtering, conditional formatting, at charts ay tumutulong sa data analysis.
