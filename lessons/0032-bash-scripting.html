Lesson 0032: Bash/Shell Scripting

Matuto ng Linux terminal basics at shell scripting para mag-automate ng system tasks.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakagamit ng basic Linux commands at makakagawa ng shell script para sa automation.

Linux Terminal Basics

Ang terminal ay text-based interface para makipag-usap sa operating system. Narito ang essential commands:

"Sa server administration, kailangan ang shell scripting — para auto backup ng database gabi-gabi."
# Navigation
ls              # List files sa current directory
ls -la          # List with details at hidden files
cd folder       # Change directory
pwd             # Print working directory

# File Operations
cp file1 file2  # Kopyahin ang file
mv file1 dir/   # Ilipat o i-rename
rm file.txt     # Delete file
mkdir newdir    # Gumawa ng folder

# Permissions
chmod 755 script.sh   # Gawing executable
chown user:group file # Palitan ang owner

# Search
grep "keyword" file.txt  # Maghanap ng pattern sa file

Shell Scripts

Ang shell script ay text file na may series ng commands. Ang shebang (#!/bin/bash) ang nagpapakilala na ito ay bash script.

#!/bin/bash
# Ito ay isang shell script

# Variables
name="Juan"
echo "Hello, $name!"

# Conditionals
if [ -f "$1" ]; then
    echo "File $1 exists."
else
    echo "File $1 does not exist."
fi

# Loop
for file in *.txt; do
    echo "Processing $file..."
done

Automating System Tasks

Pwedeng mag-schedule ng shell scripts gamit ang cron (Linux) o Task Scheduler (Windows).

#!/bin/bash
# backup.sh — Simple backup script
backup_dir="/backups/$(date +%Y%m%d)"
mkdir -p "$backup_dir"
cp -r /important/data "$backup_dir"
echo "Backup completed: $backup_dir"
