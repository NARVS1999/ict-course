Lesson 0036: HTML5 Fundamentals

Matuto ng HTML5 — ang backbone ng web. Document structure, text elements, links, images, tables, semantic HTML, at forms.

Lesson Goal Sa pagtatapos ng araling ito, ikaw ay makakagawa ng isang kumpletong HTML page gamit ang semantic elements at forms.

Document Structure

Ang bawat HTML document ay may basic structure: DOCTYPE declaration, html tag, head at body.

"HTML ang skeleton ng website — nagbibigay ng structure."
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First Page</title>
</head>
<body>
  <h1>Hello, World!</h1>
  <p>Ito ang first HTML page ko.</p>
</body>
</html>

Text Elements at Lists

<h1>Main Heading</h1>
<h2>Subheading</h2>
<p>Paragraph of text.</p>
<strong>Bold text</strong>
<em>Italic text</em>

<ul>
  <li>Unordered item</li>
  <li>Another item</li>
</ul>

<ol>
  <li>First item</li>
  <li>Second item</li>
</ol>

Links, Images, at Tables

<!-- Links -->
<a href="https://example.com">Click here</a>

<!-- Images -->
<img src="photo.jpg" alt="Description">

<!-- Tables -->
<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Juan</td>
    <td>20</td>
  </tr>
</table>

Semantic HTML

Ang semantic elements ay may kahulugan sa browser at search engines. Mas maganda para sa SEO at accessibility.

<header>Page header</header>
<nav>Navigation links</nav>
<main>Main content</main>
<article>Independent content</article>
<section>Related content group</section>
<aside>Sidebar content</aside>
<footer>Page footer</footer>

Forms

<form action="/submit" method="post">
  <label>Name:</label>
  <input type="text" name="name">

  <label>Email:</label>
  <input type="email" name="email">

  <button type="submit">Send</button>
</form>

Mini-Project: Hobby Page

Gumawa ng HTML page tungkol sa iyong paboritong hobby. Gamitin ang semantic elements (header, nav, main, footer), magdagdag ng image, isang list, at isang link sa reference site.

Summary

  • Ang HTML ay gumagamit ng tags para i-structure ang content.
  • Ang semantic elements ay nagbibigay ng meaning sa code.
  • Ang forms ay ginagamit para mag-collect ng user input.
