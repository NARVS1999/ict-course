# ICT Glossary (Taglish)

Isang diksyunaryo ng mga importanteng ICT terms sa Taglish para sa course na ito.

## Computer Hardware

**CPU (Central Processing Unit)**:
Ang utak ng computer — nagpe-process ng instructions.
_Iwasan_: Processor lang

**RAM (Random Access Memory)**:
Short-term memory ng computer — temporary storage ng data habang ginagamit.
_Iwasan_: Memory lang

**SSD (Solid State Drive)**:
Long-term storage na mas mabilis kaysa HDD — walang moving parts.
_Iwasan_: Hard drive (kung SSD)

**HDD (Hard Disk Drive)**:
Traditional storage na may spinning platter — mas mabagal pero mas mura per GB.

**Motherboard**:
Main circuit board na nagkokonekta ng lahat ng components.

**PSU (Power Supply Unit)**:
Nagbibigay ng kuryente sa lahat ng components ng computer.

**GPU (Graphics Processing Unit)**:
Nagha-handle ng graphics at visual output — important para sa gaming at video editing.

**BIOS/UEFI**:
Firmware na nag-start sa computer at naglo-load ng operating system.

**POST (Power-On Self-Test)**:
Diagnostic test na ginagawa ng computer kapag binuksan para icheck kung working ang hardware.

## Operating Systems

**OS (Operating System)**:
Software na nagmamanage ng hardware at nagpapatakbo ng applications (Windows, macOS, Linux).

**Kernel**:
Core ng operating system — direct na nagco-communicate sa hardware.

**File System**:
Paraan ng pag-organize at pag-store ng files sa storage device (NTFS, FAT32, ext4).

**CLI (Command Line Interface)**:
Text-based interface para mag-interact sa computer gamit ang commands.

**GUI (Graphical User Interface)**:
Visual interface na may windows, icons, menus, at pointers.

## Networking

**IP Address**:
Unique identifier ng bawat device sa network — parang address ng bahay.

**DNS (Domain Name System)**:
Nagsasalin ng domain names (e.g., google.com) sa IP addresses.

**DHCP (Dynamic Host Configuration Protocol)**:
Automatically nag-aassign ng IP address sa devices sa network.

**NAT (Network Address Translation)**:
Nagpapahintulot sa maraming devices na mag-share ng isang public IP address.

**LAN (Local Area Network)**:
Network na nagko-connect ng computers sa limited area (bahay, office).

**WAN (Wide Area Network)**:
Network na sumasaklaw ng malaking geographical area — ang internet ay WAN.

**Router**:
Device na nagruroute ng data packets sa pagitan ng networks.

**Switch**:
Device na nagco-connect ng devices sa loob ng isang LAN.

**Firewall**:
Security system na nagfa-filter ng incoming at outgoing network traffic.

**VPN (Virtual Private Network)**:
Gumagawa ng encrypted tunnel sa pagitan ng device at remote server para sa privacy.

**OSI Model**:
Conceptual framework na may 7 layers para maintindihan ang network communication.

**TCP/IP**:
Set ng protocols na ginagamit ng internet para mag-communicate ang devices.

**MAC Address**:
Permanenteng hardware identifier ng network interface card.

## Programming

**Algorithm**:
Step-by-step na procedure para malutas ang isang problema.

**Variable**:
Container para sa data na pwedeng magbago habang tumatakbo ang program.

**Data Type**:
Uri ng data (integer, string, boolean, float) na pwedeng i-store sa variable.

**Function**:
Reusable block ng code na gumagawa ng specific task.

**Loop**:
Structure na umuulit ng code hanggang sa ma-meet ang condition.

**Conditional**:
Statement na nag-e-execute ng code based sa condition (if/else).

**Array/List**:
Collection ng items na naka-store sa iisang variable.

**Module**:
File na naglalaman ng Python definitions at statements na pwedeng i-import.

**Git**:
Version control system na nagti-track ng changes sa code.

**Repository**:
Folder na may lahat ng files at history ng isang project na naka-version control.

## Web Development

**HTML (HyperText Markup Language)**:
Language na ginagamit para i-structure ang content ng website — skeleton.

**CSS (Cascading Style Sheets)**:
Language na ginagamit para i-style ang HTML elements — damit ng website.

**JavaScript**:
Programming language na nagbibigay ng interactivity sa website.

**HTTP/HTTPS**:
Protocol na ginagamit para mag-transfer ng data sa web.

**URL (Uniform Resource Locator)**:
Address ng isang resource sa web.

**Frontend**:
Bahagi ng website na nakikita at ini-interact ng user.

**Backend**:
Bahagi ng website na nasa server — nagpro-process ng data at nagma-manage ng database.

**API (Application Programming Interface)**:
Set ng rules para mag-communicate ang iba't ibang software applications.

**CMS (Content Management System)**:
Platform para madaling gumawa at mag-manage ng website content (e.g., WordPress).

**Responsive Design**:
Paraan ng pag-design ng website na umaangkop sa iba't ibang screen sizes.

## Databases

**Database**:
Organized collection ng data na naka-store at naa-access electronically.

**DBMS (Database Management System)**:
Software na nagma-manage ng databases (MySQL, PostgreSQL, SQLite).

**SQL (Structured Query Language)**:
Language para mag-query at mag-manage ng relational databases.

**Table**:
Collection ng related data na naka-organize sa rows at columns.

**Primary Key**:
Unique identifier para sa bawat record sa isang table.

**Foreign Key**:
Field sa isang table na nagli-link sa primary key ng ibang table.

**CRUD**:
Four basic operations: Create, Read, Update, Delete.

**JOIN**:
SQL operation na nagco-combine ng data mula sa dalawa o higit pang tables.

**Normalization**:
Process ng pag-organize ng data para mabawasan ang redundancy.

**ERD (Entity-Relationship Diagram)**:
Visual diagram na nagpapakita ng entities at relationships sa database.

## Cybersecurity

**CIA Triad**:
Confidentiality, Integrity, Availability — tatlong core principles ng security.

**Malware**:
Malicious software (virus, worm, trojan, ransomware).

**Phishing**:
Fraudulent attempt na makuha ang sensitive info sa pamamagitan ng pagpapanggap.

**Encryption**:
Process ng pag-convert ng data sa coded form para hindi mabasa ng unauthorized.

**MFA (Multi-Factor Authentication)**:
Security method na nangangailangan ng dalawa o higit pang verification factors.

**SQL Injection**:
Attack na nag-i-inject ng malicious SQL code sa queries.

**XSS (Cross-Site Scripting)**:
Attack na nag-i-inject ng malicious scripts sa web pages.

**Penetration Testing**:
Authorized simulated attack para mahanap ang vulnerabilities.

**Ransomware**:
Malware na nag-e-encrypt ng files at humihingi ng ransom para ma-decrypt.

**Social Engineering**:
Manipulation ng tao para magbigay ng confidential information.

## Cloud & Emerging Tech

**Cloud Computing**:
On-demand na access sa computing resources (servers, storage, databases) sa internet.

**IaaS (Infrastructure as a Service)**:
Cloud service na nagbibigay ng virtualized computing resources.

**PaaS (Platform as a Service)**:
Cloud service na nagbibigay ng platform para mag-develop at mag-deploy ng apps.

**SaaS (Software as a Service)**:
Cloud service na nagbibigay ng software sa internet (e.g., Google Docs).

**Virtualization**:
Technology na nagpapahintulot sa maraming virtual machines sa isang physical server.

**Container**:
Lightweight, standalone package na may code at dependencies (Docker).

**IoT (Internet of Things)**:
Network ng physical devices na may sensors at internet connectivity.

**Machine Learning**:
Subset ng AI na nagpapahintulot sa systems na matuto at mag-improve from experience.

## Productivity & Professional

**Version Control**:
System na nagre-record ng changes sa files para ma-access ang previous versions.

**Markdown**:
Lightweight markup language para sa formatting ng text.

**Pomodoro Technique**:
Time management method na gumagamit ng 25-minute focused work sessions.

**Capstone Project**:
Final project na nag-iintegrate ng lahat ng skills na natutunan sa course.

**Portfolio**:
Collection ng projects at achievements na nagpapakita ng skills.
